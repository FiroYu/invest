# 光学路线在芯片制造与互联中的范式转移

## 一句话判断
> 这不是"可能发生"的范式转移，而是**正在发生**。铜互连在 200G/lane 以上触及物理极限（信号2米内衰减），光替代铜从数据中心机架向芯片内部推进，2026年是CPO（共封装光学）量产元年。光子计算（用光而非电做计算）也已从实验室走向商业化。整条价值链从材料、芯片、封装到系统都在被重塑。

## 为什么重要

- **人类影响**：AI算力增长的最大瓶颈已从"计算"转向"数据搬运"（互联）。光学互联是突破这一瓶颈的唯一物理路径，直接决定AGI何时到来
- **经济影响**：硅光子市场 2025年~$2.9B → 2030年~$9.7B（CAGR 29.5%）；CPO市场 2024年$46M → 2030年$8.1B（CAGR 137%）。中国厂商主导全球光模块市场（70%+份额），是少数中国具有全球定价权的科技领域
- **时间敏感度**：2026年是关键拐点——台积电硅光子迈向量产，NVIDIA开始CPO量产，1.6T光模块开始部署。2027-2028年渗透率加速

## 范式转移的三个层次

本议题涵盖三条互相交织但节奏不同的技术路线：

### 层次一：光刻技术演进（芯片制造）
- **本质**：从 EUV 到 High-NA EUV，是连续演进而非范式转移
- **关键玩家**：ASML（垄断）、ZEISS（光学系统）
- **时间线**：High-NA EUV 2026年量产后用于 sub-2nm 制程
- **判断**：延续性创新，不构成结构性价值链重塑

### 层次二：光学互联替代铜（芯片互联）★ 核心范式转移
- **本质**：铜互连 → 光互连，从机架级向芯片级推进
- **演进路径**：可插拔光模块 → CPO（共封装光学）→ Optical I/O（芯片间光互连）→ 片上光互连
- **驱动力**：AI训练集群规模爆炸（十万卡+），铜缆在200G/lane以上物理失效
- **时间线**：2026年CPO量产元年，2028-2030年CPO渗透率达35%

### 层次三：光子计算（用光做计算）
- **本质**：用电做矩阵运算 → 用光做矩阵运算，潜在的终极范式转移
- **进展**：Lightmatter 2025年在Nature发表论文验证光子AI加速器
- **时间线**：商业化早期阶段（2025-2028），大规模应用需更长时间
- **判断**：高潜力但不确定性大，值得跟踪但尚非投资主线

## 核心争议点

### 正方（光替代铜已成必然）
- 铜在 200G/lane 时信号仅能传输 ~2 米，AI训练集群需要万卡互联，铜已无法胜任
- NVIDIA/AMD/Intel 三巨头联合投资 Ayar Labs，产业共识已形成
- 2026年台积电硅光子量产，CPO从概念走向产品
- 中国政策明确支持（工信部《电子信息制造业2025-2026年稳增长行动方案》）

### 反方（铜仍在反击）
- AEC（有源电缆）和 CPC（共封装铜）技术在延长铜的寿命
- CPO 的封装良率和热管理仍是挑战
- 可插拔光模块生态成熟，短期内性价比仍优于 CPO
- 光子计算的大规模商业化尚需时日

### 当前共识
- 机架间光互连已是定局（已全面光子化）
- 芯片间光互连（CPO/Optical I/O）是下一个确定性方向
- 片上光计算是远景方向，短期投资价值有限
