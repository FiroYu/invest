## 受损标的

### 一、传统CRO(商业模式受冲击)

| 公司 | 代码 | 风险逻辑 |
|------|------|----------|
| Charles River Labs | CRL | 传统动物实验/毒理学CRO，AI减少动物实验需求 |
| IQVIA | IQV | 临床CRO巨头，AI可能压缩临床试错成本 |
| LabCorp | LH | 传统检测CRO，AI预测减少送检需求 |
| Covance(LabCorp旗下) | — | 早期药物发现CRO，AI直接替代 |

**冲击路径**：AI药物发现 → 候选分子质量提升 → 临床前/临床实验次数减少 → CRO订单量下降 → CRO被迫转型提供AI辅助服务

**但需注意**：短期内CRO不会大幅受损，因为(1)AI药物发现规模仍小、(2)临床执行仍需CRO、(3)大型CRO也在投资AI能力

### 二、传统试错式药物发现

| 类别 | 说明 |
|------|------|
| 高通量筛选(HTS) | 传统方法命中率0.01-0.1%，AI虚拟筛选1-10%，HTS需求将下降 |
| 传统药物化学 | 经验驱动的分子优化，将被AI生成+预测辅助取代部分工作 |
| 动物模型实验 | AI预测毒性和药代动力学，减少动物实验需求 |

## 产业链价值重构

```
传统药物发现价值链：
靶点发现(5%) → 苗头化合物(10%) → 先导化合物(15%) → 临床前(20%) → 临床I-III期(40%) → 审批上市(10%)

AI重构后价值链：
AI靶点发现(8%) → AI分子设计(12%) → AI ADMET预测(10%) → 压缩的临床前(12%) → AI优化临床试验(45%) → 审批上市(13%)

关键变化：
- 早期发现效率大幅提升(总成本下降50-70%)
- 临床试验阶段因精准患者分层，成功率从12%提升至25-30%(预期)
- 总研发周期从10年缩短至3-5年(激进估计)或5-7年(保守估计)
```

## 投资策略

| 阶段 | 策略 | 标的 |
|------|------|------|
| 2025-2026(验证期) | 低仓位观察，关注临床数据读出 | RXRX小仓、大药企(NVS/LLY)更安全 |
| 2026-2027(确认期) | 数据积极则加仓AI平台 | RXRX、SDGR、2228.HK |
| 2028+(规模化期) | 全产业链布局 | AI平台+CRO转型赢家+空间生物学 |

## 风险提示

1. **临床失败风险**：AI设计的药物可能在临床阶段失败，证伪整个叙事
2. **估值泡沫**：多数AI Biotech无盈利，PS>20x，回调风险大
3. **大药企自建AI能力**：如果大药企自建成功，AI平台公司的合作价值下降
4. **监管不确定性**：FDA/EMA对AI辅助药物审评标准尚未完全明确
5. **数据质量瓶颈**：AI模型训练依赖高质量数据，生物数据噪声大、标注成本高
