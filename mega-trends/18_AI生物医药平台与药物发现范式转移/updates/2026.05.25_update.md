# 2026.05.25 定期更新

## 新增信息

### 重大事件
- **Insilico Rentosertib 发表 Nature Medicine**：AI 全流程发现药物（靶点+分子）的**首个**随机对照 IIa 期阳性结果。TNIK 抑制剂，IPF 治疗，60mg 组 FVC 改善 +98.4mL（安慰剂 -20.3mL）。从靶点发现到 IIa 期仅约 4 年
- **Isomorphic Labs 完成 21 亿美元 B 轮**（5月12日）：Thrive Capital 领投，累计融资 ~27 亿。IsoDDE 平台准确度比 AlphaFold 3 提升一倍以上。首次人体试验预计 2026 年底前
- **Eli Lilly-Insilico 合作**（3月30日）：首付 $1.15 亿 + 总价值 $27.5 亿，覆盖多款口服创新药
- **VantAI 连续两笔大交易**：Blueprint Medicines（总里程碑最高 $16.7 亿）+ Halda Therapeutics（总价值超 $10 亿）
- **Relay Zovegalisib 获 FDA 突破性疗法认定**：首个泛突变选择性 PI3Kα 抑制剂，Phase 3 进行中
- **Generate Biomedicines GB-0895 Phase 3**：AI 设计抗体，从发现到 Phase 3 仅 ~5 年（传统 10-15 年）

## 牛方论点（更新）

1. **Nature Medicine 发表是里程碑**：Insilico Rentosertib 是 AI 全流程发现药物的首个 IIa 期阳性结果，且发表于顶刊（IF=58.7），学术认可度极高
2. **资本加速涌入确认范式**：Isomorphic $21 亿 + Lilly-Insilico $27.5 亿 + VantAI $26.7 亿 = 大药企和顶级 VC 已从"验证概念"转向"押注管线"
3. **管线密度爆发：15 个 Phase III**：2026-2027 年将密集读出 Phase III 数据，这是范式是否成立的终极验证
4. **AI 从"理解蛋白质"跨越到"创造蛋白质"**：OpenCRISPR-1、NovaIscB、锌指蛋白设计 = 全新功能蛋白的 AI 生成已实现
5. **FDA 政策框架成型**："Good AI Practice"指南已发布，监管路径正在制度化

## 熊方论点（更新）

1. **Phase III 是药物失败的主战场**（~50% 失败率）：AI 优势主要在分子设计和毒性预测，选择正确靶点和预测临床结果方面尚无证据超越传统方法
2. **尚无 AI 药物完成 FDA 审批**：所有进展仍在"临床验证"阶段，一个 Phase III 失败可能严重打击市场信心
3. **Isomorphic 推迟一年**：首次人体试验从"2025 年底"推迟到"2026 年底"，说明 AI 药物发现的实际时间线可能比预期更长
4. **交易结构反映观望**：以"低首付 + 高里程碑"为主，大药企仍在等待临床验证

## 对抗讨论结论

- **牛方在数据上占据优势**：Nature Medicine 发表 + 173+ 管线 + 80-90% Phase I 成功率 = 范式转移正在被临床数据验证
- **但终极考验在 Phase III**：15 个 Phase III 项目 2026-2027 年读出，任何一个失败都不会否定范式，但如果多个失败则会严重动摇信心
- **蛋白质设计是新增长极**：从 AlphaFold 预测到 AI 生成全新功能蛋白，这条线的投资机会可能比药物发现更直接
