# 议题24：AI生物医药平台 — 时间线(2020-2030)

## 2020-2022：AlphaFold突破期

| 时间 | 事件 |
|------|------|
| 2020.11 | DeepMind AlphaFold 2在CASP14竞赛中达到原子级精度(GDT>92)，震惊结构生物学界 |
| 2021.07 | AlphaFold蛋白质结构数据库发布，预测35万个蛋白质结构 |
| 2021.07 | Recursion完成$2.25亿D轮融资，估值$40亿，成为AI-first Biotech标杆 |
| 2022.05 | Isomorphic Labs(DeepMind子公司)成立，专注AI药物发现 |
| 2022.07 | AlphaFold数据库扩展至2亿个蛋白质结构，覆盖几乎所有已知蛋白质 |
| 2022.09 | 英矽智能INS018_055(抗纤维化小分子)进入临床II期，首个AI端到端设计的药物 |
| 2022.11 | Schrödinger(SDGR)与BMS扩大合作，里程碑付款可达$26亿 |

| 时间 | 事件 |
|------|------|
| 2023.01 | Recursion收购Cyclica和Valence，扩展AI药物发现平台能力 |
| 2023.05 | Recursion与Nvidia战略合作，获$5000万投资，构建世界上最大的AI药物发现超算 |
| 2023.07 | AlphaFold 2开源权重发布，学术界和工业界全面拥抱 |
| 2023.09 | Eli Lilly以$4.25亿首付款签约Isomorphic Labs，总价可达$37亿 |
| 2023.12 | Recursion管线中REC-994(脑血管畸形)临床II期中期数据积极 |
| 2024.05 | AlphaFold 3发布：可预测蛋白质-配体复合物、蛋白质-核酸复合物，突破药物发现核心瓶颈 |
| 2024.06 | Novartis以$10亿+里程碑收购Mariana Oncology(放射性药物+AI靶点发现) |
| 2024.08 | Tempus AI(TEM→AI)IPO，AI+精准医学平台，市值约$60亿 |
| 2024.09 | 晶泰科技(2228.HK)港股IPO，AI+机器人药物发现 |

## 2025-2027：临床验证期(当前阶段)

| 时间 | 事件 |
|------|------|
| 2025.Q1 | 多个AI-first管线进入临床II/III期读出窗口期 |
| 2025.Q2 | Recursion与NVS合作管线首批临床数据读出 |
| 2025 | Isomorphic Labs与Eli Lilly合作项目推进至IND阶段 |
| 2025-2026 | 英矽智能INS018_055 II期数据读出——AI端到端设计的"试金石" |
| 2026 | 空间生物学技术(Spatial Biology)大规模应用于临床试验生物标志物 |
| 2026-2027 | AI设计药物首次获得FDA批准(如实现，将确立范式转移) |

## 2028-2030：范式确立期(预期)

| 时间 | 事件 |
|------|------|
| 2028 | AI药物发现市场规模突破$200亿 |
| 2029 | 大药企AI管线占比超过30% |
| 2030 | 药物发现平均周期缩短至5年以下；AI药物发现成为标准范式 |

## 关键观察节点

- **2025-2027**：临床II/III期数据密集读出——决定AI药物发现是"真突破"还是"过度承诺"
- **2026**：首个AI设计药物获批可能性最高的时间窗口
- **2028**：市场共识是否从"验证期"切换到"规模化期"
