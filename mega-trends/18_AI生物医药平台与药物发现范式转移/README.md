# 议题18：AI生物医药平台与药物发现范式转移

## 核心判断

AI药物发现从"概念验证"进入"临床验证"阶段。AlphaFold 3可预测蛋白质-配体复合物，AI-first Biotech管线进入临床II/III期密集读出期(2025-2027)。十年$500亿市场。药物发现从10年/$26亿缩短至3-5年/$5-10亿。2025-2027年是"证明或证伪"关键窗口。

## 关注模型

- AI设计药物的临床成功率(vs传统制药的历史12%)
- **Phase III数据密集读出(2026H2-2027)** — 15个项目，终极验证窗口
- 大药企AI合作签约量与首付款规模（2026 H1已达>$75亿总潜在价值）
- **蛋白质设计从预测→创造的产业化进展**（AlphaProteo/Baker Lab/Tencent AI）
- CRO业务模式转变程度(从实验执行→AI辅助设计)
- AI Biotech平台管线密度与推进速度
- **FDA AI药物审评框架正式化进度**（实时临床试验试点已启动）
