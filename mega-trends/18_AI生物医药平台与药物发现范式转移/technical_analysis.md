# 议题24：AI生物医药平台 — 技术分析

## 一、AI药物发现技术栈

AI药物发现覆盖从靶点到临床的完整链条，各环节技术成熟度差异显著：

```
靶点发现 → 分子设计 → ADMET预测 → 临床试验设计 → 生物标志物发现
  [成熟]     [成熟]     [中等]       [早期]         [新兴]
```

### 1. 靶点发现(Target Discovery)

- **技术**：多组学数据整合(基因组学、蛋白质组学、代谢组学) + 知识图谱 + 因果推理
- **代表**：Recursion的表型筛选平台(每月生成数百万个细胞图像数据)、BenevolentAI的知识图谱
- **突破点**：AlphaFold 3使结构靶点发现从"实验验证"转向"计算预测"
- **局限**：靶点验证(确认靶点与疾病因果关系)仍需湿实验

### 2. 分子设计(Molecular Design)

- **生成式AI**：扩散模型(Score-based generative models)、Transformer架构用于分子生成
- **虚拟筛选**：从数十亿化合物库中筛选候选分子，速度提升100-1000倍
- **代表**：Schrödinger的FEP+(自由能微扰)精确预测分子结合亲和力
- **关键指标**：命中率(Hit Rate)从传统HTS的0.01-0.1%提升至AI方法的1-10%

### 3. ADMET预测

- **ADMET**：吸收(Absorption)、分布(Distribution)、代谢(Metabolism)、排泄(Excretion)、毒性(Toxicity)
- **技术**：图神经网络(GNN)预测分子性质、多任务学习同时预测多个ADMET参数
- **价值**：提前淘汰"不可能成药"分子，减少动物实验和早期临床失败
- **挑战**：训练数据有限(尤其是人体内数据)，预测准确度仍需提升

### 4. 临床试验设计

- **Digital Twin临床试验**：基于患者历史数据构建虚拟对照组，减少对照组人数
- **适应性试验设计**：AI实时分析中期数据，动态调整剂量和入组标准
- **患者分层**：通过基因组学和生物标志物精准筛选应答人群
- **代表**：Tempus的AI平台整合200万+患者数据优化临床试验

### 5. 生物标志物发现

- **空间生物学(Spatial Biology)**：在组织原位解析基因表达的空间分布
- **技术**：10x Genomics Visium、MERFISH、CODEX
- **价值**：发现组织微环境中的预测性生物标志物，精准定位药物应答人群

## 二、AlphaFold 3的能力与局限

### 能力

| 能力 | 说明 |
|------|------|
| 蛋白质-配体复合物预测 | 可预测小分子药物如何结合蛋白质靶点——药物发现的核心问题 |
| 蛋白质-核酸复合物 | 可预测蛋白质与DNA/RNA的相互作用，开拓RNA药物设计 |
| 多链蛋白质复合物 | 可预测抗体-抗原复合物、蛋白质二聚体等 |
| 翻译后修饰 | 可模拟磷酸化、糖基化等修饰对结构的影响 |
| 精度 | 蛋白质-配体对接精度接近实验方法(中位RMSD < 2Å) |

### 局限

| 局限 | 说明 |
|------|------|
| 动态构象 | 预测静态结构，蛋白质的动态构象变化(dynamics)仍是挑战 |
| 膜蛋白 | 跨膜蛋白(重要药物靶点)预测精度较低 |
| 别构位点 | 对非活性位点(别构位点)的预测不够可靠 |
| 共价结合 | 无法预测共价抑制剂与靶点的结合模式 |
| 溶剂效应 | 未充分考虑水分子在配体结合中的作用 |

### 对药物发现的影响

- **靶点可成药性评估**：从"实验探索"变为"计算筛选"，大幅降低早期失败率
- **苗头化合物发现**：基于结构的虚拟筛选从辅助手段升级为主要手段
- **但**：AlphaFold解决的是"结构"问题，不是"化学"问题——分子优化仍需实验迭代

## 三、空间生物学(Spatial Biology)技术原理

### 核心概念

空间生物学在保持组织空间结构的前提下，同时测量数千个基因的表达水平。传统单细胞测序丢失了细胞的空间位置信息，空间生物学弥补了这一缺陷。

### 主要技术路线

| 技术 | 公司 | 原理 | 通量 | 空间分辨率 |
|------|------|------|------|-----------|
| Visium | 10x Genomics | 空间条形码捕获mRNA | 全转录组 | 55μm(多细胞) |
| MERFISH | Vizgen(被10x收购) | 原位杂交+多重荧光成像 | 500+基因 | 亚细胞(100nm) |
| Xenium | 10x Genomics | 原位测序 | 5000+基因 | 亚细胞 |
| CODEX | Standard BioTools | 抗体条形码+循环成像 | 40+蛋白 | 亚细胞 |
| CosMx | Standard BioTools | 原位杂交成像 | 6000+基因 | 亚细胞 |

### 应用场景

- 肿瘤微环境解析：识别免疫细胞浸润的空间模式
- 神经科学：脑区功能与基因表达的映射
- 药物发现：发现预测药物应答的组织空间生物标志物

## 四、Digital Twin临床试验概念

### 原理

利用历史患者数据构建虚拟患者模型，作为临床试验的"数字化对照组"，从而：
- 减少对照组患者数量(伦理和经济优势)
- 加速试验数据读出
- 预测试验结果

### 技术栈

1. 患者数据整合(EHR + 基因组 + 影像 + 可穿戴设备)
2. 疾病进展建模(常微分方程/深度学习)
3. 治疗效果预测(因果推断)
4. 虚拟对照组生成(合成控制法)

### 代表公司

- **Unlearn.AI**：与FDA合作推进Digital Twin试验设计
- **Tempus(AI)**：200万+患者数据库支撑临床试验优化
- **Recursion(RXRX)**：表型数据驱动的患者分层

### 监管进展

- FDA 2023年发布"Digital Twin临床试验指南"草案
- EMA正在评估Digital Twin对照试验的科学有效性
- 预计2026-2027年首批获批案例

### 平台类型分类

```
AI药物发现平台
├── 数据驱动型：Recursion(图像)、Tempus(临床数据)
├── 结构驱动型：Isomorphic Labs(AlphaFold)、Schrödinger(物理计算)
├── 生成式AI型：英矽智能(Insilico Medicine)
└── 自动化驱动型：晶泰科技(XtalPi)——AI设计+机器人执行
```

### 投资逻辑差异

- **Recursion(RXRX)**：管线最丰富，临床验证最先到来，但烧钱率高($3亿+/年)
- **Schrödinger(SDGR)**：SaaS模式+合作收入，现金流更好，但增长较慢
- **Tempus(AI)**：收入最大，但偏"数据服务"而非"药物发现"
- **晶泰科技(2228.HK)**：中国市场AI制药龙头，政策利好，但港股流动性差
- **英矽智能(ISOL)**：端到端故事最完整，但临床阶段最早，风险最高
