# 更新日志 — 2026.05.21 初始研究记录

## 议题建立：mRNA癌症疫苗 — 个性化医疗范式

### 研究背景

本议题建立于 2026 年 5 月 21 日，旨在系统性追踪 mRNA 癌症疫苗从临床验证到商业化的全过程。mRNA-4157 Phase 3 数据预计在 2026 年下半年至 2027 年初读出，这是决定整个领域命运的关键事件。

### 当前状态总结

**已确认的事实：**
- mRNA-4157（Moderna/Merck）5年 Phase 2 数据显示复发/死亡风险降低 49%（HR 0.51）
- 已获 FDA 突破性疗法认定（黑色素瘤辅助治疗）
- Phase 3 V940-001 试验正在进行中，全球多中心招募
- BioNTech 有 20+ 临床肿瘤项目，多平台并行
- 全球有 557 项 mRNA 临床试验在进行中（ClinicalTrials.gov）
- Moderna 正在将管线从黑色素瘤扩展至 NSCLC 等更多适应症

**待验证的关键假设：**
- [ ] mRNA-4157 Phase 3 是否达到主要终点（无复发生存期）
- [ ] Phase 3 的 HR（风险比）是否与 Phase 2 的 0.51 一致或更优
- [ ] BioNTech BNT111 是否启动 Phase 3
- [ ] Moderna NSCLC 项目 Phase 2/3 是否启动并产生积极早期数据
- [ ] 个性化制造周期是否从 4-6 周缩短至 3-4 周
- [ ] FDA 是否基于 Phase 3 阳性数据给予加速审批
- [ ] 支付方（保险公司/Medicare）对个性化疫苗的报销态度
- [ ] mRNA 罕见病疗法（mRNA-3927）是否显示功能性治愈潜力

### 关键数据缺口

1. **Phase 3 中期分析数据**：2026年H2 是否会有中期分析（取决于事件数）
2. **NSCLC 扩展数据**：Moderna 是否公布 NSCLC 的早期临床数据
3. **制造优化进展**：个性化周期缩短的具体数据
4. **竞争者动态**：CureVac、Arcturus 等是否在肿瘤领域有所突破
5. **定价讨论**：FDA 咨询委员会或支付方是否有公开讨论
