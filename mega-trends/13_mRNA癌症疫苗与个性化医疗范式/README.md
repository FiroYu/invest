## 核心判断

mRNA 技术正在从"疫苗平台"进化为"可编程的药物开发操作系统"。COVID 疫情验证了 mRNA 平台的快速响应能力（从序列到临床仅 63 天），现在这一能力正在被指向癌症——人类最大的疾病负担。

**范式转变**：从"一药治百人"到"一人一方"。个性化 mRNA 癌症疫苗为每位患者的肿瘤新抗原（neoantigen）量身定制 mRNA 序列，教会免疫系统识别和攻击癌细胞。

**核心证据**：Moderna/Merck 的 mRNA-4157（V940）联合 Keytruda 用于黑色素瘤的 5 年随访数据显示：
- 复发/死亡风险降低 **49%**
- 无远处转移生存期显著延长
- 已获 FDA 突破性疗法认定
- Phase 3 正在进行中，数据 2026-2027 年读出

如果 Phase 3 验证 Phase 2 结果，这将是**个性化 mRNA 癌症疫苗的历史性突破**——不只是新药，而是新范式。

与此同时，mRNA 平台的应用边界持续扩展：罕见病蛋白质替代、自身免疫病、基因编辑递送、慢性给药。mRNA 正在成为继小分子和抗体之后的**第三大药物模态**。
