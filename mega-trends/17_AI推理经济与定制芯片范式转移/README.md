# 议题17：AI推理经济与定制芯片范式转移

## 核心判断

AI投资从"训练军备竞赛"转向"推理规模战争"。LLM token使用量年增250%+。定制ASIC(Broadcom/Google TPU/Amazon Inferentia)从通用GPU夺取推理负载，ASIC市场$120亿→$300亿+(2027)。先进封装(CoWoS/3D)成为比GAA更关键的瓶颈。推理是"持续运营成本"而非"一次性CapEx"，决定AI商业可行性。

|------|------|

## 范式转移逻辑

### 从训练到推理
- **训练**是一次性CapEx：训练GPT-4级别模型约$1亿+，但训练完成后即固定
- **推理**是持续OpEx：每次用户调用模型都产生计算成本，且随用户量线性/超线性增长
- 随着AI应用落地，推理计算量已远超训练计算量（估计10:1到100:1）
- Token使用量2026年1月到4月增长250%（6.4万亿→22.7万亿），反映推理需求爆发

### 从通用GPU到定制ASIC
- **NVIDIA GPU**(A100/H100/B200)：训练+推理通用，CUDA生态强大，但推理效率不是最优
- **定制ASIC**：为特定推理场景优化，能效比提升3-5倍，成本降低50-70%
- Hyperscaler（Google/AWS/Microsoft/Meta）都有自研芯片计划，降低对NVIDIA依赖
- Broadcom为多家Hyperscaler设计定制ASIC，收入从$80亿→$300-400亿(2027)

### 先进封装成为真正瓶颈
- CoWoS(Chip on Wafer on Substrate)是TSMC的先进封装技术
- 当前月产能~3万片，但需求>5万片，严重供不应求
- 先进封装的瓶颈比先进制程(GAA)更关键——即使有设计能力，没有封装产能也无法量产

## 关注模型

- NVIDIA推理收入占比变化（确认范式转移的速度）
- ASIC设计中标数量（Broadcom/Marvell等的新订单）
- CoWoS产能利用率和扩产计划（TSMC月度数据）
- Token单价趋势（$/million tokens）——推理成本下降速度决定应用落地速度
- 各Hyperscaler自研芯片的量产时间表
- HBM产能与价格（SK Hynix/Samsung/Micron）

## 反共识视角：GPU算力从稀缺到过剩

市场共识是GPU永远不够——但2027-2028可能转向过剩。定制ASIC效率提升3-5x + 推理优化(量化/蒸馏/稀疏) + 推理专用芯片(Groq/Cerebras)量产 → 有效算力供给增长可能超过需求增长。GPU租赁价格已开始下降(H100从$8/GPU-hr降至$2-3)。如果过剩兑现，NVIDIA估值承压，但推理成本加速下降利好AI应用层。

**关键信号**：GPU集群利用率<70%持续一季度 → 确认过剩拐点

## 潜在风险

- NVIDIA CUDA护城河比预期更强，ASIC迁移速度慢于预期
- AI应用变现不达预期，导致推理需求增速放缓
- 先进封装产能瓶颈缓解后，竞争加剧
- 地缘政治影响TSMC产能（议题21关联）
- ASIC的设计-流片-量产周期长（2-3年），可能出现技术迭代风险
