## 核心判断

口服GLP-1于2026年1月上市，这是减肥药市场的"功能机→智能手机"时刻。注射壁垒的消除将GLP-1从医疗场景推向大众消费场景，整条食品/饮料/餐饮/健康价值链正在被结构性重塑。

**范式转移的核心数据：**
- 用户卡路里摄入降低 20-30%
- 含糖饮料消费降低 65%
- 酒精消费降低 62%
- 家庭杂货支出在用药6个月内下降 5.3%（Cornell大学研究证实）

## 争议与不确定性

### 专利悬崖节奏
- **基础semaglutide化合物专利**：2026年在加拿大和中国到期，仿制药可进入
- **美国市场**：通过制剂专利、给药设备专利（Novo的注射笔）延长至约2040年
- **影响**：非美市场可能出现低价仿制药，挤压Novo利润但扩大用户基数

### 保险覆盖博弈
- 雇主保险覆盖范围尚未普遍改善，80%雇主将GLP-1列为医疗成本上升的首要驱动因素
- Medicare/Medicaid覆盖仍有限制，需要额外的适应症（如心血管风险降低）才能获得更广泛报销
- 价格是关键变量：口服剂型能否降低定价是扩大覆盖的前提

### 长期安全性
- 肌肉流失风险：快速减重伴随瘦体重下降，长期代谢影响仍在研究中
- 胃肠道副作用：恶心、呕吐、腹泻在用药初期普遍存在
- 长期使用数据（>5年）仍在积累中
- 停药反弹：部分研究显示停药后体重回升，终身用药的依从性和成本问题
