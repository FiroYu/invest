# GLP-1减肥药 — 技术分析

## 1. GLP-1受体激动剂作用机制

### 生理基础
GLP-1（胰高血糖素样肽-1）是一种肠促胰岛素激素，由肠道L细胞在进食后分泌。其生理作用包括：
- **促进胰岛素分泌**：葡萄糖依赖性刺激胰岛β细胞
- **抑制胰高血糖素**：降低肝脏葡萄糖输出
- **延缓胃排空**：减少进食量，增加饱腹感
- **作用于中枢神经系统**：直接抑制食欲中枢（下丘脑弓状核）
- **心血管保护**：改善内皮功能、降低炎症标志物

### 靶点组合演进

| 代际 | 靶点组合 | 代表药物 | 机制优势 |
|------|---------|---------|---------|
| 单靶点 | GLP-1 | Semaglutide (Wegovy) | 成熟机制，心血管获益明确 |
| 双靶点 | GLP-1 + GIP | Tirzepatide (Zepbound) | GIP增强脂肪氧化，协同减重 |
| 三重靶点 | GLP-1 + GIP + Glucagon | Retatrutide | Glucagon增加能量消耗，减重天花板最高 |
| 新组合 | GLP-1 + 胰淀素(Amylin) | Amycretin | 胰淀素增强饱腹感，口服潜力 |

**关键洞察**：从单靶点到三重靶点，减重效果从~15%提升至~24%，逼近减重手术（~25-30%）的效果。

## 3. 下一代技术路线

### 3.1 口服小分子
- **意义**：消除注射壁垒，GLP-1从医疗场景进入大众消费场景
- **技术路线**：
  - **口服肽类**：Novo的口服semaglutide利用SNAC吸收促进剂，需空腹服用
  - **口服小分子**：Lilly的Orforglipron是非肽类小分子，不受食物限制，服药便利性更高
- **市场影响**：口服剂型可能将可及人群扩大3-5倍

### 3.2 三重激动剂
- **Retatrutide**（Lilly）：GLP-1+GIP+Glucagon三重靶点
- **Glucagon的附加价值**：增加基础代谢率、促进脂肪氧化、增加能量消耗
- **减重天花板**：~24%，已接近减重手术效果（~25-30%）

### 3.3 长效制剂
- **目标**：从每周注射延长至每月甚至每季度
- **Amgen MariTide**：基于抗体偶联技术，目标每月一次给药
- **意义**：进一步降低给药负担，提高依从性

### 3.4 组织选择性
- **下一代方向**：在保持减重效果的同时减少肌肉流失（瘦体重保护）
- **策略**：组合GLP-1与肌肉保护剂（如myostatin抑制剂）
- **市场意义**：解决"减肥但虚弱"的担忧，对老年用户尤为关键

## 4. 专利分析

### Semaglutide（Novo Nordisk）

| 专利类型 | 覆盖范围 | 到期时间 |
|---------|---------|---------|
| 基础化合物专利 | 全球 | 2026年（加拿大/中国已到期）；欧洲约2026-2030 |
| 制剂专利（注射用制剂） | 美国 | ~2032年 |
| 给药设备专利（注射笔） | 美国 | ~2040年 |
| 口服制剂专利（SNAC技术） | 美国 | ~2035-2040年 |

**关键结论**：
- **非美市场**：基础化合物专利2026年开始到期，仿制药可进入加拿大、中国等市场
- **美国市场**：通过制剂和设备专利将保护期延长至~2040年，Novo在美国的核心利润池受到保护
- **战略意义**：Novo的策略是在美国维持高利润的同时，在非美市场通过品牌和口服剂型保持竞争力

### Tirzepatide（Eli Lilly）

| 专利类型 | 覆盖范围 | 到期时间 |
|---------|---------|---------|
| 基础化合物专利 | 美国 | ~2036年 |
| 制剂专利 | 美国 | ~2036-2040年 |

Lilly的专利保护期较Novo更长，且tirzepatide作为双靶点分子有更强的差异化壁垒。

## 5. 对食品行业的量化影响

### 消费行为变化数据（基于Morgan Stanley研究 + Cornell研究）

| 消费品类 | GLP-1用户消费变化 | 传导机制 |
|---------|------------------|---------|
| 总卡路里摄入 | -20% ~ -30% | 中枢性食欲抑制 |
| 含糖饮料 | -65% | 奖赏回路改变，甜食欲望下降 |
| 酒精 | -62% | 同上，GLP-1作用于中脑奖赏系统 |
| 家庭杂货支出 | -5.3%（6个月内） | Cornell大学实证研究 |
| 快餐/外出就餐 | 显著下降（估算-15~25%） | 食量减少+对高热量食物兴趣降低 |
| 高蛋白食品 | 需求增加 | 医生/营养师推荐高蛋白饮食保护肌肉 |

### 食品公司已采取的应对措施
- **雀巢（Nestle）**：推出GLP-1友好型产品线（Vital Pursuit），高蛋白、小份量
- **Conagra**：在财报电话会中讨论GLP-1影响，调整产品组合
- **PepsiCo / Coca-Cola**：加速向零糖/功能性饮料转型
- **超市/零售**：Kroger、Walmart开始设置GLP-1友好货架标签
