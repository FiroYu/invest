# 2026.05.25 定期更新

## 新增信息

### 重大事件
- **Retatrutide Phase 3 数据**（5月22日）：三重激动剂80周减重28.3%，2年最高30.3%（相当于减重手术效果），但仅比替尔泊肽多3%，低于市场预期
- **Foundayo（Orforglipron口服）已上市**：4月1日 FDA批准（50天审批，2002年以来最快），$149/月，首批数天超1,000张处方
- **ATTAIN-MAINTAIN 试验**：注射转口服可维持78-82%减重成果 → 用户可能长期处于"食欲抑制"状态
- **CagriSema 挫折**：Phase 3 减重23%未击败替尔泊肽25.5%，Novo股价跌15%
- **加拿大首批仿制药上市**：司美格鲁肽仿制药5月上市，定价降至品牌药35-50%

### 市场与影响数据
- **GLP-1 活跃用户超900万**（美国），预计5年内达3,000-5,000万
- **食品行业冲击加速**：GLP-1家庭杂货支出-5.3%，含糖饮料-65%。加拿大GLP-1渗透率从4-5%升至8%（一年翻倍），食品经济年损失$23-34亿加元
- **酒精行业**：用户酒精摄入-62%，美国饮酒人口从62%降至54%
- **市场规模**：2025年$450-640亿，2032-2034年$1,100-1,900亿（CAGR 10-14%）

### 供应链
- Eli Lilly 累计投入超$150亿扩产（阿拉巴马$60亿+苏州$30亿+神户）
- 口服 GLP-1 供应链远比注射剂简单（标准口服固体制剂线），若口服占比提升，部分注射剂产能投资可能"搁浅"

## 牛方论点（更新）

1. **口服 GLP-1 已上市**：Foundayo + Wegovy片 = "功能机→智能手机"时刻已兑现。无空腹要求的 Foundayo 用药依从性更高
2. **ATTAIN-MAINTAIN 证明长期用药**：注射转口服维持78-82%减重 → 用户不会停药，食品/酒精行业的影响是永久性的
3. **渗透率加速**：加拿大一年从4-5%到8%，口服进一步降低门槛，美国5年内可能达3,000-5,000万用户
4. **仿制药加速渗透**：加拿大首批上市，定价35-50%，将进一步扩大用户基础

## 熊方论点（更新）

1. **Retatrutide 效果低于预期**：三重激动剂仅比双靶点多3% → "机制叠加"边际收益递减，下一代管线天花板可能比想象低
2. **CagriSema 输给替尔泊肽**：Novo下一代管线不确定性上升，竞争格局可能向 Lilly 倾斜
3. **BALANCE Model 无限期延迟**：Medicare 覆盖短期无法大规模扩展，限制老年人群渗透
4. **Trump 降价协议限制定价权**：Medicare/Medicaid $245/月，患者自付$50/月

## 对抗讨论结论

- **"功能机→智能手机"判断已兑现**：口服 GLP-1 上市是范式转移的里程碑
- **但下一代管线天花板可能低于预期**：Retatrutide +3% 和 CagriSema 受挫说明"机制叠加"收益递减
- **食品/酒精行业影响是永久性的**：ATTAIN-MAINTAIN 证明用户不会停药，这是做空食品/酒精的最强证据

## 辩论评级

- **评级：维持「通过」**
