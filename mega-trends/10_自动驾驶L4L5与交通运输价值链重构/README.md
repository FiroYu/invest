# 议题10：自动驾驶L4L5与交通运输价值链重构

## 核心判断

2026年是自动驾驶出行商业化拐点年。Waymo在11个城市部署~3,700+辆车，提供50万+次/周付费出行，证明Robotaxi模式可规模化。Aurora的无人卡车实现10条路线商业运营，获得McLane（Berkshire Hathaway）超级客户验证。Ojai第六代车型将单车成本降低$75,000，Waymo盈利路径加速清晰。WeRide/Uber联手进军马德里，中国Robotaxi公司集体出海中东和欧洲。法律责任正在从个人驾驶员系统性转移向制造商，Morningstar预测个人汽车保险可能在20年内过时。

## 关注模型

- **Robotaxi单量/覆盖城市增速**：衡量商业化的核心速度指标。Waymo每季度新增城市数和单量增长率直接反映扩展节奏。
- **自动驾驶卡车运营里程/收入**：Aurora月度运营里程和收入是B2B场景商业化的硬指标。
- **Ojai部署速度**：Waymo第6代车型的逐月部署量是判断其盈利路径的最重要指标。
- **Uber AV GMV占比**：Uber平台上自动驾驶出行的占比是"平台赢家"论点的核心验证。
- **个人汽车保险费率变化**：当保险公司开始基于自动驾驶数据调整费率模型时，标志着责任转移进入实质性阶段。
