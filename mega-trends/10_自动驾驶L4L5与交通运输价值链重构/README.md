# 议题10：自动驾驶L4L5与交通运输价值链重构

## 核心判断

2026年是自动驾驶出行商业化拐点年。Waymo在10个城市部署3700+辆车，已提供约50万次付费出行，证明Robotaxi模式可规模化。Aurora的无人卡车在德州实现24/7不间断商业运营，将自动驾驶最直接的商业价值——替代长途卡车司机——变为现实。法律责任正在从个人驾驶员系统性转移向制造商，Morningstar预测个人汽车保险可能在20年内过时。这一转变不仅影响汽车行业，将重塑保险、物流、城市规划和能源基础设施。

## 关键指标

| 指标 | 当前值 | 趋势 |
|------|--------|------|
| Waymo运营城市 | 10个 (2026.3) | 持续扩展 |
| Waymo车队 | 3700+辆 | 快速增长 |
| Waymo付费出行 | ~50万次 | 商业化 |
| 无人驾驶卡车 | 200+辆预计2026上路 | 从0到1 |
| Baidu Apollo覆盖 | ~20个城市 | 中国领先 |
| Tesla Robotaxi | 奥斯汀运营中 | 扩展到达拉斯/休斯顿 |

## 关注模型

- **Robotaxi单量/覆盖城市增速**：衡量商业化的核心速度指标。Waymo每季度新增城市数和单量增长率直接反映扩展节奏。
- **自动驾驶卡车运营里程**：Aurora、Kodiak等公司的月度运营里程是B2B场景商业化的硬指标。
- **个人汽车保险费率变化**：当保险公司开始基于自动驾驶数据调整费率模型时，标志着责任转移进入实质性阶段。
- **Uber/Lyft司机收入趋势**：司机供给和收入的变化是Robotaxi替代效应的滞后指标，但方向性意义重大。
