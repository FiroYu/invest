# 初始研究记录 — 2026.05.21

## 研究背景

电网储能议题是"能源转型"大主题下的核心子议题。选择在此时间点建立跟踪框架的原因：

1. **成本临界点已过**：LFP电池 $52/kWh 意味着4小时储能在主要市场已比天然气调峰便宜，经济性拐点已确认
2. **部署加速**：2025年全球部署110GW，年增速翻倍，从"有意义"变为"主导性"
3. **负电价扩散**：德国475小时、加州12%白天负电价，表明可再生能源过剩已成系统性现象
4. **长时储能突破在即**：Form Energy铁-空气电池2027-2029年商业化，可能彻底改变储能格局

## 数据来源

- BloombergNEF (BNEF)：电池成本数据、储能部署统计
- Lazard：LCOE/LCOS分析
- IRENA：可再生能源成本报告
- Wood Mackenzie：储能市场预测
- Form Energy：铁-空气电池技术白皮书和商业进展
- CAISO/ENTSO-E：电力市场负电价数据

## 关键假设

1. LFP电池成本将继续下降，2027年达到 $30-40/kWh
2. Form Energy铁-空气电池能在2027-2029年成功商业化
3. 政策环境（IRA、中国储能补贴）不会发生重大逆转
4. 电网接入瓶颈将逐步缓解（虽然可能限制短期增速）

## 待深入研究的问题

1. 中国储能项目的实际利用率（部分项目存在"建而不用"现象）
2. 钠离子电池的产业化进展和实际成本竞争力
3. 美国储能项目的并网排队问题和解决方案
4. 储能安全标准对部署速度的影响
5. 长时储能（8小时+）的经济性拐点何时到来
6. 储能对电力市场设计的影响（容量市场、辅助服务市场改革）
