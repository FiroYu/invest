# 2026.05.21 — 初始建立

## 议题背景

全球供应链碎片化与多极再工业化议题由三位不同背景分析师独立提出，增强了议题的可信度和重要性。三位分析师分别从地缘政治、产业经济、金融市场角度观察到同一结构性趋势。

## 核心观察

1. **范式转移确认**：全球贸易从"效率最优"转向"安全优先"，这不是周期性波动而是结构性重构
2. **三条主线并行**：美国再工业化、友岸外包、中国产能出海同时推进，互相加强
3. **速度超预期**：美国制造业建设支出+220%、墨西哥FDI创历史纪录、印度iPhone占比达70%
4. **转口贸易复杂性**：Stanford研究表明中国通过越南/墨西哥转口仍保持影响力，真实转移程度需要更仔细分析

## 待深入研究

- [ ] 各国制造业产能建设的实际投产率（规划 vs 实际）

- [ ] 自动化技术（机器人、AI制造）能否缩小劳动力成本差距的时间线
- [ ] 关税政策的进一步演变对供应链成本的边际影响
- [ ] 中国跨境电商"四小龙"在不同市场的合规风险
