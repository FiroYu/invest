# 议题16：全球供应链碎片化与多极再工业化

## 核心判断

全球贸易从"效率最优"转向"安全优先"，驱动三条并行主线——美国再工业化(CHIPS Act/IRA)、友岸外包(墨西哥/越南/印度)、中国产能出海。持续10-15年的结构性重构。三位不同背景分析师独立提出同一议题，增强可信度。

## 三条并行主线

### 主线1：美国再工业化
- CHIPS Act：$520亿半导体制造补贴，吸引台积电、三星、Intel在美建厂
- IRA（通胀削减法案）：$3,690亿清洁能源与制造业税收抵免
- 国防供应链本土化要求（国防授权法案）
- 制造业建设支出从2019年$700亿飙升至2025年$2,300亿+

### 主线2：友岸外包(Friend-shoring)
- 墨西哥：103个在建工业园、$61.4亿投资，2025年FDI达$400-410亿历史纪录
- 越南：FDI $276.2亿(2025)，对美智能手机出口占30%+
- 印度：PLI（生产挂钩激励）计划，70%在美销售iPhone已为印度制造
- 东欧（波兰、捷克）承接欧洲近岸外包

### 主线3：中国产能出海
- 储能出海订单366GWh，同比+144%
- 跨境电商"四小龙"：SHEIN(估值$600亿+)、Temu(PDD)、TikTok Shop、速卖通
- 新能源产业链海外建厂（宁德时代匈牙利基地、比亚迪泰国/巴西工厂）
- 中国通过东南亚、墨西哥"转口"保持全球供应链影响力（Stanford研究）

## 关注模型

- 各国FDI流向与制造业产能建设速度
- 关税政策变化与贸易壁垒演进
- 跨境电商GMV增速与海外仓布局
- 美国制造业就业数据回流速度
- 中间品贸易流向（判断真实供应链转移 vs 转口贸易）

### 反共识视角：印度="第二个中国"电子制造
印度不只是服务外包，正在复制中国2000年代的电子制造崛起路径。iPhone印度组装从2%→14%，PLI(生产挂钩激励)补贴$260亿吸引富士康/纬创/TSMC考虑建厂。印度电子制造目标$500B(2030)。关键差异：印度有14亿人口的国内市场+英语优势+民主制度对西方友好。如果印度电子制造复合增速维持30%+，2028-2030年将成为供应链格局的第三极。

## 潜在风险

- 再工业化成本高昂，美国制造业时薪$35+ vs 墨西哥$5、越南$3
- 地缘政治突变可能导致供应链二次重构
- 中国出口管制升级对全球产业链的冲击
- 逆全球化推高全球通胀中枢1-2个百分点
