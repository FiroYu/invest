# 议题20：日本通缩终结与30年资产重定价

## 核心论点

日本"通缩叙事"已终结。2025年春斗工资增长5.1%（30年最高），BOJ结束负利率+YCC，核心CPI稳定在2.5%+。这不是周期性反弹，而是30年一遇的regime change。

工资-通胀螺旋如果确立，将触发连锁反应：日本国债收益率上行（从0.8%→2%+）、日本保险/银行股重估、日元升值（从150→120-130）、日本国内消费复苏。风险在于BOJ紧缩过快可能刺破日本国债市场。

## 反直觉角度

30年的通缩思维意味着大多数全球投资者对日本的结构性低配是系统性的。这是自1990年以来最大的regime change。当一种叙事持续了30年，它会嵌入资产定价的每一个角落——从日本国债的期限溢价（几乎为零）到日本企业的定价行为（习惯性不涨价）到家庭资产配置（现金+存款占54%）。一旦这个叙事反转，调整将是全面的、非线性的。

**类比**：这就像2012年的日本股市——所有人都在做空，但安倍经济学改变了游戏规则。这一次，改变的不是政策预期，而是通胀现实本身。

## 关注板块

### 一级受益：日本银行与保险
- **逻辑**：利率上行直接扩大净息差（NIM），保险公司持有的国内债券组合浮盈改善
- **银行**：三菱UFJ(8306.T)、三井住友FG(8316.T)、瑞穗(8411.T)
- **保险**：东京海上(8766.T)、三井住友海上(8769.T)、日本生命(7189.T)

### 二级受益：国内消费复苏
- **逻辑**：工资增长5%+意味着实际可支配收入提升，消费意愿从"省钱"转向"花钱"
- **标的**：Fast Retailing(9983.T)、Seven & i(3382.T)、ZOZO(3579.T)

### 三级受益：日本房地产
- **逻辑**：通胀环境下实物资产保值需求上升，日元升值吸引外资买入日本不动产
- **标的**：三井不动产(8801.T)、三菱地所(8802.T)

### 对冲考量：出口商
- **风险**：日元升值侵蚀出口商利润，需关注汇率对冲情况
- **标的**：丰田(7203.T)、索尼(6758.T)——优选有汇率对冲管理的公司

## 关切模型（Concern Model）

### 上行催化剂（确认信号）
- [ ] 春斗工资增长连续第二年>4%
- [ ] BOJ明确释放继续加息信号（>0.75%）
- [ ] 日本核心CPI连续6个月>2.5%
- [ ] 日经与TOPIX背离收窄（内需股补涨）
- [ ] 日本家庭现金配置比例降至52%以下
- [ ] 外资连续3个月净买入日本内需板块

### 下行风险（证伪信号）
- [ ] 日本GDP连续两个季度负增长
- [ ] 10Y JGB收益率突破2.5%引发国债市场动荡
- [ ] 日元急速升值至110以下（伤害出口）
- [ ] 春斗工资增长回落至3%以下
- [ ] 核心CPI跌回1.5%以下
- [ ] BOJ被迫重新转向宽松

### 灰天鹅事件
- 日本国债市场流动性危机（BOJ持有超过50%的JGB）
- 日本政府债务/GDP超过260%引发评级下调
- 全球衰退导致日本出口断崖式下跌
