# Issue 30：关键时间线与政策路径

## 历史脉络——日本通缩30年

### 通缩时代的起点（1990-2000）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1989.12 | 日经225指数触及38,915点历史高点 | 泡沫顶点 |
| 1990-1992 | 日本股市暴跌60%，地价开始下跌 | 资产泡沫破裂 |
| 1995 | 银行危机全面爆发，住专问题浮出水面 | 金融系统风险暴露 |
| 1997-1998 | 亚洲金融危机 + 日本消费税上调（3%→5%） | 经济二次探底 |
| 1999.02 | BOJ实施零利率政策（ZIRP） | 首次零利率实验 |

### "失去的二十年"（2000-2012）

| 时间 | 事件 | 意义 |
|------|------|------|
| 2001.03 | BOJ首创量化宽松（QE） | 全球央行QE先驱 |
| 2003-2007 | 小泉结构改革，不良贷款清理 | 短暂复苏但未解决通缩 |
| 2006.03 | BOJ结束零利率，加息至0.25% | 首次尝试正常化——失败 |
| 2008-2009 | 全球金融危机，日经跌破7,000 | 外部冲击打断正常化 |
| 2010.10 | BOJ推出全面宽松政策 | 重回宽松 |
| 2012.12 | 安倍晋三上台，推出"三支箭" | 安倍经济学开始 |

### 安倍经济学到通缩终结（2012-2025）

| 时间 | 事件 | 意义 |
|------|------|------|
| 2013.04 | 黑田东彦就任BOJ行长，推出QQE | "两年内实现2%通胀" |
| 2016.01 | BOJ实施负利率政策（-0.1%） | 负利率时代开始 |
| 2016.09 | 引入收益率曲线控制（YCC），10Y目标0% | YCC框架建立 |
| 2018-2019 | 消费税上调（8%→10%），经济放缓 | 政策收紧尝试受阻 |
| 2020.03 | COVID冲击，日经一度跌破16,000 | 外部冲击 |
| 2022.03 | 全球通胀飙升，核心CPI突破2% | 日本通胀首次"达标" |
| 2023.01 | BOJ将YCC上限从0.25%上调至0.5% | 政策转向第一步 |
| 2023.07 | YCC上限上调至1.0% | 加速转向 |
| 2023.10 | YCC上限实质上调至1.0%+，灵活化 | 名义维持、实质放松上限 |
| 2024.03 | BOJ结束负利率，加息至0-0.1% | **历史性转向：负利率终结** |
| 2024.07 | BOJ加息至0.25%，减少购债 | 第二次加息 |
| 2024.08 | 日元急剧升值，日经单日暴跌12%（8月5日） | 市场剧烈波动 |
| 2025.03 | 春斗结果：工资增长5.1%（30年最高） | **通缩终结的关键确认信号** |
| 2025.05 | BOJ加息至0.5%，继续缩减购债 | 正常化持续推进 |
| 2025.Q3 | 新NISA制度满18个月，资金流入加速 | 家庭资产配置转变 |

## 政策路径推演（2026-2028）

### 情景一：有序正常化（概率 55%）

```
2026 Q1-Q2  →  BOJ加息至0.75%
              10Y JGB收益率缓升至1.2-1.5%
              日元升至135-140

2026 Q3-Q4  →  BOJ加息至1.0%
              10Y JGB收益率1.5-1.8%
              日元升至125-130
              内需股开始补涨

2027        →  BOJ利率稳定在1.0-1.5%
              10Y JGB收益率1.8-2.2%
              工资增长维持4%+
              家庭资产从存款向投资转移加速
```

### 情景二：过快紧缩引发动荡（概率 25%）

```
2026 H1     →  BOJ过快加息至1.0%+
              10Y JGB收益率突破2.0%
              JGB市场流动性恶化
              日元急速升值至115-120

2026 H2     →  出口商利润大幅下修
              日本GDP负增长
              BOJ被迫暂停甚至回调利率
              日经回调20-30%
```

### 情景三：外部冲击打断正常化（概率 20%）

```
全球衰退 / 地缘冲突 → 日本出口断崖
                     → CPI回落至1.5%以下
                     → BOJ暂停加息
                     → 日元重回150+
```

## 关键监测日历

| 日期 | 事件 | 关注点 |
|------|------|--------|
| 每月第三周五 | BOJ货币政策会议 | 利率决定、前瞻指引措辞变化 |
| 每月中旬 | 日本CPI公布 | 核心CPI是否维持2%+ |
| 每季度首月 | 短观报告（Tankan） | 企业通胀预期、资本支出计划 |
| 每年3月 | 春斗最终结果 | 工资增长确认 |
| 每季度 | GDP初值 | 实际经济增长是否跟上通胀 |
| 每月20日 | JGB拍卖 | 债券市场需求情况 |
| 每季度 | NISA资金流入数据 | 家庭资产配置转变进度 |
