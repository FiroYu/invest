# 议题12：EV固态电池与汽车石油价值链重构

## 核心判断

电池成本正以超预期速度下降。Goldman Sachs 预测 2026 年电池包成本降至 **$80/kWh**，届时 EV 在多数细分市场将达到与燃油车的成本平价（无需补贴）。中国龙头 BYD 2025 年全球销量 **226 万辆**，已超越 Tesla（164 万辆），标志着中国 EV 在规模和成本上同时取得全球领先。

固态电池技术将于 **2027-2028 年**进入商业化阶段（Toyota 领先），实现 1200km 续航、10 分钟快充，从根本上消除续航焦虑。与此同时，全球石油需求可能已在 **2025-2026 年**见顶——BloombergNEF 和 IEA 模型均显示，EV 替代效应将在 2030 年前每天减少 5-10 百万桶石油需求。

这一议题追踪的不是单一技术突破，而是一个**系统性价值链迁移**：从石油→电力、从内燃机→电机、从机械→软件、从西方品牌→中国制造。

### 反共识视角：石油供给峰值比需求峰值更重要
市场聚焦"石油需求见顶"（EV替代），但真正的冲击可能在供给端。OPEC闲置产能2026年降至历史低位，美国页岩油产量增速放缓（资本纪律+甜点带枯竭），上游勘探投资存在$380B/年缺口(IEA)。即使需求温和增长，2027-2028年可能面临供给不足，油价冲$120+。这意味着：石油多头可能同时从供给和需求两端获得支撑（短期供给紧张 + 长期需求替代），石油股的价值重估被市场低估。

## 关注模型

- **电池 $/kWh 成本曲线**：行业最核心的单一指标，决定 EV 经济性拐点
- **BYD/Tesla 季度交付量**：全球 EV 竞争格局的直接体现
- **全球 EV 渗透率**：S 曲线增长，关注各国拐点
- **全球石油需求趋势**：是否已过峰值，IEA vs OPEC 预测分歧
- **OPEC闲置产能与美国页岩产量**：供给端约束的早期预警
- **固态电池商业化时间表**：从实验室到量产的里程碑追踪
- **各国 EV 关税/补贴政策**：贸易壁垒 vs 成本优势的博弈

## 研究范围

- 电池技术路线图（LFP → 钠离子 → 固态 → 铁-空气）
- EV 制造商竞争格局（中国 vs 西方 vs 日本/韩国）
- 石油需求结构性下降的时间表和幅度
- 固态电池商业化的技术和商业里程碑
- 关税政策对中国 EV 出海的影响和绕行策略
- 内燃机供应链的价值侵蚀路径
