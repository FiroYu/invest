# AI架构范式转移 — 经济影响与投资分析

## 场景假设

以下分析基于 **JEPA 在 2027-2028 年成为 AI 主流架构** 的假设。若此假设不成立（LLM 继续主导），则大部分"受损"判断不成立。

## 受益行业

| 行业 | 逻辑 | 受益程度 | 代表公司/标的 |
|------|------|----------|---------------|
| **边缘AI芯片** | JEPA 1B参数可在边缘运行，推理不再依赖云端 | 高 | 高通、联发科、苹果、瑞芯微、全志科技 |
| **NPU/ASIC设计** | JEPA 运算更规则，专用芯片性价比远超 GPU | 高 | 寒武纪、地平线、爱芯元智、Graphcore |

| **自动驾驶** | 世界模型直接提升物理推理能力 | 高 | 特斯拉FSD、Waymo、小鹏、华为车BU |

## 受损行业

| 行业 | 逻辑 | 受损程度 | 代表公司/标的 |
|------|------|----------|---------------|
| **高端GPU（训练侧）** | 不再需要万卡集群训练，H100/B200 需求可能结构性下降 | 高 | NVIDIA（训练业务部分） |
| **数据中心基础设施** | 算力需求降10-100x，在建数据中心可能过剩 | 高 | Equinix、Digital Realty、万国数据 |
| **HBM（高带宽内存）** | JEPA 推理显存需求仅2-4GB，不再需要 TB 级 HBM | 高 | SK海力士、三星(内存)、美光 |
| **InfiniBand/高速互联** | 训练集群从万卡缩至百卡，互联需求骤降 | 高 | NVIDIA(Mellanox)、Arista Networks |
| **核电/数据中心供电** | AI电力需求预期下调，新建核电项目可能推迟 | 中 | Constellation Energy、Vistra |
| **液冷散热** | 功耗下降后传统风冷可能重新够用 | 中 | Vertiv、CoolIT |
| **LLM API服务商** | 模型可在本地运行，API 调用量可能下降 | 中 | OpenAI、Anthropic、Cohere |
| **大模型训练数据集** | JEPA 样本效率更高，对"海量文本"的需求降低 | 低-中 | Common Crawl、The Pile 等 |

## 基础设施变化

### 芯片
- **从 GPU 优先到 NPU/ASIC 多元化**：不再只有 NVIDIA 一种选择，专用芯片的价值凸显
- **训练端**：从万卡 A100/H100 集群 → 百卡甚至单机训练
- **推理端**：从数据中心 → 边缘设备（手机/汽车/IoT）
- **制程压力缓解**：1B 模型不需要 3nm，成熟制程即可

### 存储
- **权重文件**：从 TB 级 → GB 级
- **KV Cache**：不再需要（非自回归推理）
- **训练数据**：从万亿 token → 千万级视频/多模态样本
- **整体**：存储行业增长逻辑可能从"AI驱动"转向其他驱动因素

### 电力
- **训练**：从 GPT-4 级 ~50GWh → 可能降低 10-100x
- **推理**：从数据中心 MW 级 → 瓦级（边缘设备）
- **AI 电力需求峰值可能提前见顶**

### 网络
- **训练集群内部**：不再需要 InfiniBand 级互联
- **推理 API 流量**：大幅下降（本地推理为主）
- **但多模态数据传输**：视频训练数据可能增加带宽需求

## 投资策略

### 短期（0-6个月）— 观察期
- **核心动作**：跟踪 JEPA 在语言生成上的进展，这是最大不确定性
- **观察指标**：Meta 是否发布 language-JEPA；Google/Apple 是否有 JEPA 相关论文/专利
- **可操作**：关注 V-JEPA 2 的实际部署案例（机器人、自动驾驶）

### 中期（6-24个月）— 布局期
- **若信号确认**（JEPA 在语言任务取得突破）：
  - 增持：边缘 AI 芯片、NPU 设计公司、机器人/自动驾驶
  - 减持：HBM、高端数据中心 REIT、InfiniBand 设备商
  - 关注：存算一体初创公司（可能成为下一个被大厂收购的对象）
- **若信号证伪**（LLM 仍主导）：
  - 维持当前 AI 基础设施投资逻辑不变

### 长期（2年以上）— 结构性变化
- **AI民主化**：不再需要百万美元训练成本，中小公司可训练自己的世界模型 → AI 应用层竞争加剧
- **地缘影响**：如果 JEPA 降低了对先进制程和高端 GPU 的依赖，芯片出口管制的效力可能下降
- **电力结构**：AI 数据中心电力需求增速放缓，但机器人/自动驾驶的大规模部署可能创造新的电力需求

## 关键风险提示

1. **以上所有投资分析都建立在 JEPA 成为主流的假设上**，当前仍处于早期验证阶段
2. LLM 生态的惯性极强（开发者工具、企业部署、用户习惯），即使 JEPA 技术更优，商业替代也需要更长时间
3. NVIDIA 等巨头不会坐以待毙，可能通过收购、自研或适配来维持地位
4. "更优架构"不等于"商业胜利"——历史上 Betamax、Windows Phone 等都是技术更优但商业失败的案例
