# 议题15：网络安全战与数字基础设施防御

## 核心判断

Volt Typhoon（伏特台风）已被美国CISA、FBI、NSA联合确认——国家支持的攻击者已预置在美国关键基础设施系统中（能源、水务、通信、交通），目的是在战时实施破坏。这标志着网络安全从"IT预算项目"转变为"国家安全支出"的范式转移。

全球网络安全市场2025年规模 $219-255B，预计2030年增长至 $335-580B（CAGR 7-15%）。国防网络安全领域增速更为迅猛，从2025年 $20.35B增长到2033年 $54.46B（CAGR约13%）。关键基础设施保护市场从2025年 $154B增长到2030年 $197B。

这一议题的核心问题是：**国家级网络攻击从"间谍活动"升级为"战时破坏准备"，将如何重塑全球网络安全投资格局？**

## 关键指标

| 指标 | 当前值 | 趋势 |
|------|--------|------|
| 全球网络安全市场 | $219-255B (2025) | 2030年$335-580B |

| 勒索软件攻击频率 | 每11秒一次 (2024) | 持续增加 |
| 零信任市场规模 | $38B (2025) | 2030年$100B+ |

## 关注模型

需要持续追踪的动态指标：

1. **企业网络安全预算增速** — 网络安全支出占IT总预算的比例是否从当前5-15%向20%+移动
2. **国家级网络攻击频率** — APT组织活动频率、攻击复杂度演变
3. **保险行业网络风险定价** — 网络保险费率、承保条件变化，反映风险的量化
4. **监管合规要求升级** — NIS2、DORA、SEC网络安全披露等法规的实施影响
5. **关键基础设施暴露面** — IT/OT融合进度、SCADA系统联网比例

## 投资含义

- **长期受益者**：CrowdStrike (CRWD)、Palo Alto Networks (PANW)、Fortinet (FTNT)、Zscaler (ZS)、Tenable (TENB)
- **ETF工具**：CIBR、HACK、BUG — 提供行业分散化暴露
- **关键观察标的**：Mandiant/Google Cloud的安全业务、国防网络安全承包商
- **结构性受损**：网络安全防御不足的公用事业公司、依赖传统OT系统的运营商、低估网络风险的保险公司
