## TC/RC分析（铜冶炼加工费）

### TC/RC走势

| 时期 | TC/RC（$/吨） | 意义 |
|------|---------------|------|
| 2022年 | 80-90 | 供给充裕 |
| 2023年中 | 70-80 | 开始收紧 |
| 2024年初 | 40-50 | 显著收紧 |
| 2024年末 | 20-30 | 冶炼严重亏损 |
| 2025年 | 15-25 | 历史低位 |
| 2026E | 10-20 | 极端低位 |

**TC/RC的经济含义**：当TC/RC降至$30以下时，中国铜冶炼厂开始亏损。2024-2025年已有冶炼厂宣布减产/检修。这意味着即使铜矿产出不减少，冶炼产能不足也会成为供给瓶颈。

### 中国铜冶炼厂利润模型

```
冶炼利润 = TC/RC + 硫酸收入 - 加工成本 - 铜精矿溢价

基准情景（2025年）：
TC/RC: $25/吨
硫酸收入: ~¥100/吨（硫酸价格低迷）
加工成本: ~¥200/吨
→ 冶炼利润: 接近零甚至为负

结果：冶炼厂被迫减产 → 精炼铜供给进一步收紧 → 支撑铜价
```

## 铝市场技术分析

### LME铝价

| 指标 | 数值 | 判断 |
|------|------|------|
| 当前价格 | $2,400-2,600/吨 | 中高位区间 |
| 2022年高点 | $3,870 | 距高点仍有空间 |
| 200日均线 | $2,350 | 上方运行 |
| 关键阻力 | $2,800-3,000 | 突破后看$3,500 |
| 关键支撑 | $2,200 | 成本支撑 |

### 铝的供给特性

- **产能上限**：中国电解铝产能上限4,500万吨（政策性约束）
- **能源成本**：电解铝电力成本占比40-50%，能源价格是关键变量
- **铜铝替代**：铜价>$10,000后，电力领域铜铝替代加速
- **回收率**：铝回收率高于铜（铝回收率~70% vs 铜回收率~45%）

### 铝的定价逻辑

```
铝价 = 成本支撑（电力+氧化铝+阳极） + 合理利润 + 铜价溢出效应

基准情景：
电力成本: ~¥0.4/kWh
氧化铝: ~¥3,500/吨
→ 完全成本: ¥16,000-17,000/吨（~$2,200-2,300/吨）
→ 如果铜价$12,000+，铝价有望升至$3,000-3,500
```

## 量化策略

### 季节性规律

| 月份 | 历史表现 | 原因 |
|------|---------|------|
| 1-2月 | 偏弱 | 中国春节需求淡季 |
| 3-5月 | 偏强 | 中国节后补库+春季施工 |
| 6-8月 | 震荡 | 消费淡季+检修季 |
| 9-10月 | 偏强 | 金九银十+冬储前备货 |
| 11-12月 | 混合 | 年末资金面+来年预期 |

### 铜金比

| 指标 | 当前值 | 历史均值 | 含义 |
|------|--------|---------|------|
| 铜/金价格比 | 3.8 | 5.0 | 铜相对黄金被低估 |
| 铜金比低于均值 | 经济衰退担忧 | 或者 | 黄金被高估 |

**交易思路**：如果全球经济不陷入深度衰退，铜金比有望从3.8回升至5.0，即铜相对黄金有30%的上涨空间。

### 曲线结构（期货升贴水）

| 合约 | 价格结构 | 意义 |
|------|---------|------|
| 现货 vs 3月 | 现货升水 | 现货紧张 |
| 3月 vs 15月 | 近月升水 | 近期供需偏紧 |
| 整体曲线 | Backwardation | 牛市结构 |

**历史规律**：铜处于deep backwardation时（现货>3月>$50），通常对应价格上行周期的中后段，距离顶部可能还有6-12个月。
