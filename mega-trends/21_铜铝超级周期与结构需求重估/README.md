## 核心论点

市场仍在用传统需求框架（房地产/基建）定价铜铝，但AI数据中心+EV+电网基建正在创造全新的结构性需求。这是被系统性低估的范式转变。

AI数据中心单栋用铜量是传统建筑的3-5倍（变压器+电缆+散热系统），EV单车用铜量83kg vs 燃油车23kg，全球电网升级需额外$6,000亿投资。铜的"新能源属性"被系统性低估。供给端：全球铜矿品位持续下降，新矿开发周期10-15年，2027-2028年可能出现严重供给缺口。LME铜$12,000+/吨可期。

## 反直觉角度

铜正在从"周期性大宗商品"被重新定价为"结构性增长大宗商品"。这是一次十年一遇的框架转变。

传统分析师用"中国房地产需求"作为铜价的锚，但这个锚正在失效——中国房地产用铜占比已从35%降至约20%，而AI/EV/电网用铜占比从15%快速升至30%+。当一种商品的驱动因子发生结构性变化时，基于历史周期的定价模型将系统性地低估其价格。

**核心洞察**：铜是AI时代的"隐形赢家"——市场为英伟达付60x PE，但忽略了AI数据中心每一瓦电力背后需要的铜。铜的供给弹性极低（新矿10-15年），而AI需求弹性极高（每6个月翻倍），这个不对称性是定价错误的根源。

## 关注板块

### 一级受益：铜矿生产商

- **逻辑**：铜价每涨$1,000/吨，大型铜矿企业利润增加10-20%，经营杠杆极高
- **一线**：Freeport-McMoRan(FCX)、Southern Copper(SCCO)、Antofagasta(ANTO.L)
- **二线**：Teck Resources(TECK)、First Quantum(FM.TO)、Las Bambas（五矿资源1208.HK）
- **中国标的**：江西铜业(0358.HK)、紫金矿业(2899.HK)、洛阳钼业(3993.HK)

### 二级受益：铝生产商

- **逻辑**：铝作为铜的替代品，当铜价飙升时铝的需求和价格也会跟涨
- **标的**：中国铝业(2600.HK)、中国宏桥(1378.HK)、力拓(RIO)、美铝(AA)
- **注意**：铝的供给弹性高于铜（电解铝产能可扩展），涨幅可能弱于铜

### 三级受益：回收技术与服务

- **逻辑**：铜价高企将推动废铜回收率提升，回收技术公司受益
- **标的**：Sims Limited(SGM.AX)、欧冶链金、格林美(002340.SZ)

### 四级受益：铜ETF与衍生品

- **标的**：Copper ETF(CPFX)、iPath Bloomberg Copper ETN(JJC)、LME铜期货

- [ ] 中国房地产投资进一步大幅下滑（>10%）
