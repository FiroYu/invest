# 地缘冲突与战争风险 — 事件时间线

## 长期趋势

### 2014-2022（地缘风险升温期）
- **2014**: 俄罗斯吞并克里米亚，冷战结束后欧洲安全格局首次剧变
- **2018**: 中美贸易战开始，供应链"武器化"成常态
- **2022-02**: 俄罗斯入侵乌克兰，欧洲安全秩序根本改变
- **2022-2024**: 全球军费进入持续上升通道，北约国家承诺2%GDP国防开支

### 2023-2024（供应链重构期）
- **2023**: 红海危机爆发，胡塞武装袭击商船，欧亚航运成本飙升
- **2023-2024**: 友岸外包/近岸外包加速，跨国企业采用"1+N"多点布局
- **2024**: 美国加大对中国半导体出口管制（先进制程+设备）
- **2024**: 全球FDI流量处于低位，产业链重构明显

## 2025（军备竞赛加速期）

- **2025**: 全球军费达 **$2.89万亿**（+2.9% YoY，连增10年）— [SIPRI](https://www.sipri.org/media/press-release/2026/global-military-spending-rise-continues-european-and-asian-expenditures-surge)
- **2025**: 欧洲国防支出 **$8640亿**（+14.1% YoY），首次大幅超过美国增速 — [Economist](https://www.economist.com/graphic-detail/2026/05/11/by-one-measure-americas-allies-now-outspend-it-on-defence)
- **2025**: NATO盟国军费同比增20%，全部达标 — [Atlantic Council](https://www.atlanticcouncil.org/commentary/trackers-and-data-visualizations/nato-defense-spending-tracker/)
- **2025-Hague**: NATO峰会确立新目标：**5% GDP** 用于国防+防务相关支出
- **2025-04**: Lockheed Martin 与 Rheinmetall 深化合作，共建欧洲导弹/火箭制造中心 — [Lockheed Martin](https://news.lockheedmartin.com/Lockheed-Martin-and-Rheinmetall-to-join-forces-on-centre-of-excellence-for-European-Security)
- **2025-11**: 德国联邦议院批准2026年军费 **$1290亿**（+45% YoY）
- **2025**: EU设立 **€1000亿** 国防专项基金
- **2025**: Eurasia Group发布2026十大风险：美国政治革命、大国博弈、欧洲围困、俄罗斯混合战争等 — [Eurasia Group](https://www.eurasiagroup.net/issues/top-risks-2026)
