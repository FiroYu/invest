# AI Agent / 自主AI劳动力 — 初始研究记录 (2026.05.21)

## 核心发现

1. **Copilot → Agent的范式转移已启动**：2025-2026年是拐点。Salesforce Agentforce产生实际收入、Codex Agent和Claude Code面向开发者部署、白宫量化AI对GDP的+1.3%贡献——这些信号表明AI从"辅助"进入"替代"阶段。

2. **入门级白领岗位面临结构性压力**：Dario Amodei的50%入门级岗位风险警告虽存在争议，但方向是确定的。客服、数据录入、文档处理、初级编码等岗位已经在大规模被AI Agent替代。

3. **BPO行业面临生存级威胁**：BPO的商业模式建立在"人力成本套利"之上。AI Agent消除了这种套利的基础——一个AI Agent的成本低于任何国家的人力成本。这不是边际效率提升，而是商业模式颠覆。

4. **MCP协议的标准化意义**：Anthropic推动的Model Context Protocol正在成为AI Agent连接外部工具的标准协议。如果成功，Anthropic将获得类似Microsoft在PC时代的生态影响力。

5. **企业部署从试点到生产**：2026年是企业Agent大规模进入生产环境的一年。关键场景：客服（80%+常规咨询）、代码生成（30-50%常规开发）、文档处理、数据录入。

## 交叉影响分析

### 与议题「GLP-1减肥药」（交叉）
- **AI加速药物发现**：GLP-1的快速迭代（从Wegovy到三重激动剂）部分得益于AI辅助药物设计。
- **AI+精准医疗**：个性化给药方案、临床试验优化。

### 与议题「稳定币/数字货币」（交叉）
- **AI Agent需要支付基础设施**：自主Agent之间的商业交易需要低成本、即时的支付结算，稳定币天然适配。
- **AI驱动的金融自动化**：AI Agent处理财务流程时，稳定币/数字货币比传统银行API更易集成。

### 与议题「老龄化」（交叉）
- **AI护理助手**：AI Agent在养老场景的应用（健康监测、用药提醒、社交陪伴）。
- **老龄化劳动力替代**：人口老龄化导致的劳动力短缺，AI Agent可能部分填补。

### 与议题「医疗成本」（交叉）
- **AI降低行政成本**：医院/保险的行政流程（编码、理赔、预约）被AI Agent自动化，降低运营成本。
- **AI诊断**：降低诊断错误率，减少返工和诉讼成本。
