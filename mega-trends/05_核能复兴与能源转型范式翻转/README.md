# 核能复兴与能源转型范式翻转

## 一句话判断
> 从福岛后的"弃核"到2026年的"拥核"是一次180度范式翻转。AI数据中心电力需求爆炸（2030年~1000 TWh），四大科技巨头签署10+ GW核电合同，全球127种SMR在开发，COP28首次将核能纳入气候议程。铀矿和核电运营商是确定性最高的受益者，SMR开发商是高风险高回报的投机标的。

## 为什么重要
- **人类影响**：核能是唯一能提供7x24零碳基荷电力的技术，直接决定净零排放能否实现
- **经济影响**：全球核电装机目标从当前376 GW增至2050年1,446 GW，铀矿结构性供给紧张
- **时间敏感度**：2023-2026是政策翻转的关键窗口，科技公司核电合同已锁定长期需求

## 核心争议点

### 正方（核能复兴确定）
- AI数据中心需要7x24基荷电力，只有核电匹配
- COP28/29/30连续三年核能宣言，50+国家支持
- 美国两党罕见共识：拜登$9亿SMR+特朗普核能复兴
- 科技巨头10+ GW核电合同锁定需求
- 铀矿供给不足，结构性短缺

### 反方（经济性存疑）
- NuScale首秀项目成本从$36亿飙至$93亿后取消
- SMR无大规模验证案例，建设周期不确定性大
- 核废料永久处置库全球仅芬兰1个
- 可再生能源LCOE持续下降
- 宾大研究称SMR为"昂贵的死胡同"

### 当前共识
- 核能复兴的方向确定，但SMR商业化时间表和成本仍有不确定性
- 传统核电运营商和铀矿商的确定性高于SMR开发商

### 反共识视角：增强型地热(EGS)可能是比核聚变更近的颠覆者
核聚变更性感但商业化时间表在2035+，增强型地热已经实现商业发电。Fervo Energy 2024年在犹他州实现商业规模EGS发电（3.5MW），成本降至$20/MWh，IPO预计2026-2027。地热的优势：24/7基荷（无间歇性）、无处不在（不需要铀矿）、不需要特殊燃料循环。Google已签约Fervo为数据中心供电。如果EGS成本曲线延续，2030年可能成为与核电并列的零碳基荷选项。
