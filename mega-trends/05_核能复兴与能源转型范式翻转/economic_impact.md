# 核能复兴与能源转型范式翻转 — 技术与投资分析

## 1. SMR与传统核电对比

| 维度 | 传统核电 | SMR |
|------|---------|-----|
| 单机容量 | 1,000+ MW | 10-300 MW |
| 建设周期 | 7-15年 | 目标3-5年 |
| 初始投资 | $80-150亿 | $5-30亿 |
| 安全系统 | 主动安全 | 被动安全（重力/自然对流） |
| 选址 | 受限 | 可部署在工业设施附近 |

## 2. SMR五大玩家

| 公司 | 型号 | 状态 | 商业化 |
|------|------|------|--------|
| **NuScale** | VOYGR 77MW | 最成熟设计，NRC已认证；但首秀项目因成本取消 | ~2030 |
| **Oklo** | 快中子微堆15-50MW | 已上市(NYSE:OKLO)，NRC审查中 | 2027-2028 |
| **TerraPower** | Natrium 345MW | 怀俄明煤电厂旧址建设中(Bill Gates支持) | ~2030 |
| **X-energy** | Xe-100 80MW | 政府+商业双支撑 | ~2030 |
| **Rosatom** | RITM-200 | 源自破冰船反应堆，已运行 | 已运行 |

## 3. 投资分层

### 高确定性：铀矿+核电运营商

| 标的 | 逻辑 |
|------|------|
| **Cameco (CCJ)** | 全球最大上市铀矿商，产量~2100万磅 |
| **Kazatomprom** | 全球最大铀生产商（~2910万磅/年） |
| **Constellation Energy (CEG)** | 美国最大核电运营商，Microsoft三里岛交易受益 |
| **中国核电 (601985)** | 每年核准6台机组，装机增长确定 |
| **中广核电力 (01816.HK)** | 发电量创新高 |

### 高风险高回报：SMR开发商

| 标的 | 逻辑 |
|------|------|
| **Oklo (OKLO)** | 已上市，Sam Altman支持 |
| **NuScale (SMR)** | 已上市，技术最成熟但商业前景不确定 |
| TerraPower / X-energy | 未上市 |

### 受损板块
- 煤炭（长期结构性衰退）
- 反核ESG基金（被迫调仓）

## 4. 投资时间窗口
- **2024-2027**: 预期博弈期，铀矿确定性最高
- **2027-2030**: 验证期，首批SMR投运数据决定走向
- **2030+**: 规模化期，如SMR经济性得到验证
