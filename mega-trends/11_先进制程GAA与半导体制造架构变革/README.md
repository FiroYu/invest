# 议题11：先进制程GAA与半导体制造架构变革

## 核心判断

从FinFET到GAA（Gate-All-Around，全环绕栅极）纳米片是晶体管架构自2011年FinFET引入以来最根本性的变化。背面供电（Backside Power Delivery）是另一层独立革命，将电源线从晶体管上方移到下方，释放布线空间并提升性能。TSMC N2已于2025Q4开始量产，良率约70%，Apple和NVIDIA是首批客户。ASML进入EUV超级周期——2026年60台、2027年80台标准EUV加10台高NA EUV——设备商是确定性最高的受益者。2nm晶圆价格约$30,000（5nm的2倍），但每晶体管成本仅增约5%，密度提升抵消了晶圆涨价的绝大部分。

## 关键指标

| 指标 | 当前值 | 趋势 |
|------|--------|------|
| TSMC N2量产 | 2025Q4开始 | 良率~70% |
| 2nm晶圆价格 | ~$30,000 | 是5nm的2倍 |
| 每晶体管成本 | 仅增~5% | 密度提升抵消 |
| ASML EUV出货 | 2026年60台, 2027年80+10台高NA | 超级周期 |
| 高NA EUV单价 | ~3.5亿欧元 | — |
| SK海力士EUV订单 | $79亿 | 确认需求 |
| Intel 18A | 延迟至2026 | 同时GAA+背面供电 |

## 关注模型

- **ASML季度订单/EUV出货量**：最领先的行业晴雨表。ASML的订单积压和出货节奏反映全球先进制程投资的力度和确定性。
- **TSMC先进制程营收占比**：N3+N2占TSMC营收的百分比，衡量先进制程的需求强度和客户买单意愿。
- **Intel代工良率进展**：Intel 18A的良率是Intel代工复兴的关键。如果持续低于50%，客户可能继续排队TSMC。
- **每晶体管成本趋势**：虽然晶圆价格持续上涨，但密度提升是否足以保持每晶体管成本的摩尔定律趋势，是行业可持续性的核心问题。
