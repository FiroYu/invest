# 议题13：2nm/GAA/背面供电 — 技术分析

## 一、FinFET → GAA纳米片架构变化

### FinFET的物理极限

FinFET的沟道是竖直的"鳍"（fin），栅极从三面包围沟道。随着制程缩小，鳍的宽度越来越窄，导致：
- **驱动力下降**：更窄的鳍意味着更少的电流通道
- **需要更多鳍**：为保持驱动力，需要并排多个鳍，但这增加了面积
- **寄生电容增加**：鳍之间的间距缩小带来串扰

当制程推进到3nm以下，FinFET的量子效应和寄生效应使其难以继续缩放。

### GAA纳米片的工作原理

GAA（Gate-All-Around）纳米片将沟道从竖直的"鳍"改为水平的多层纳米片（nanosheet），栅极从四面完全包围每一层沟道。

**结构变化**：
```
FinFET（侧面视图）：
    ┌─栅极─┐
    │ ┌─┐ │
    │ │鳍│ │  ← 栅极三面包围
    │ └─┘ │
    └─────┘
    硅衬底

GAA纳米片（侧面视图）：
    ┌──栅极──┐
    │ ═纳米片═│  ← 沟道水平放置
    │ ═纳米片═│  ← 多层堆叠
    │ ═纳米片═│
    └────────┘  ← 栅极四面包围
    硅衬底
```

**GAA的关键优势**：
1. **更完美的栅极控制**：四面环绕比三面控制更彻底，漏电流更低
2. **可调节沟道宽度**：不像FinFET受限于鳍的离散宽度，纳米片的宽度可以连续调节，优化驱动电流
3. **更好的面积效率**：多层堆叠在不增加面积的情况下提供更大的驱动力
4. **更好的静电特性**：短沟道效应被更好地抑制

### GAA的变体

| 变体 | 结构 | 采用者 | 特点 |
|------|------|--------|------|
| 纳米片（Nanosheet） | 宽的扁平沟道 | TSMC, Intel, Samsung | 驱动力强，适合高性能 |
| 纳米线（Nanowire） | 窄的圆柱形沟道 | Samsung（部分） | 栅极控制最佳，但驱动力较低 |
| CFET | N/P型垂直堆叠 | 下一代（2028+） | 面积效率最高，制造最复杂 |

## 二、背面供电（BSPDN）技术原理

### 传统正面供电的问题

在传统工艺中，信号线和电源线都在晶体管的同一侧（正面）。随着晶体管密度增加：
- 信号线和电源线竞争有限的布线空间
- 电源线越细，电阻越大，IR压降越严重
- 信号线的布线资源被电源线挤占

### 背面供电的解决方案

将电源网络从晶圆正面移到背面：
1. 先在正面完成晶体管和信号线的制造
2. 将晶圆翻转（flip）
3. 从背面减薄硅衬底
4. 在背面制造电源配送网络（通过TSV通孔连接到正面晶体管）

```
传统（正面供电）：          背面供电：
┌─信号线────────┐      ┌─信号线────────┐
├─电源线────────┤      │               │  ← 更多信号线空间！
├─信号线────────┤      ├─信号线────────┤
├─晶体管────────┤      ├─晶体管────────┤
└─硅衬底────────┘      ├─TSV通孔───┤
                       └─背面电源网络─┘  ← 独立电源层
```

### TSMC A16 "Super Power Rail" vs Intel PowerVia

| 特性 | TSMC Super Power Rail (A16) | Intel PowerVia (18A) |
|------|----------------------------|---------------------|
| 首次量产 | 2026年底-2027 | 2026年（18A） |
| 集成方式 | 在N2 GAA基础上加背面供电 | GAA(RibbonFET)+背面供电同时引入 |
| 电源轨密度 | 更高（Super Power Rail直接在晶体管下方） | 标准TSV通孔方式 |
| 性能提升 | 预计6-10%频率提升，IR压降改善 | 预计5-8%频率提升 |
| 制造复杂度 | 极高（需要晶圆减薄至纳米级精度） | 高（Intel称已解决关键技术挑战） |

**Intel的赌博**：在18A节点同时引入GAA（RibbonFET）和背面供电（PowerVia），是一次"双跳"——同时引入两项革命性技术。风险极高，但如果成功，可能缩小与TSMC的技术差距。

## 三、高NA EUV vs 标准EUV

### EUV光刻原理

EUV（极紫外光）使用13.5nm波长的光进行光刻，比深紫外（DUV，193nm）波长短一个数量级，可以实现更精细的图案。

### 关键参数对比

| 参数 | 标准EUV (NXE:3800) | 高NA EUV (EXE:5200) |
|------|-------------------|---------------------|
| 数值孔径（NA） | 0.33 | 0.55 |
| 分辨率 | ~13nm | ~8nm |
| 放大倍率 | 4x | 8x（Anamorphic） |
| 视场尺寸 | 26mm x 33mm | 26mm x 16.5mm（减半） |
| 单台价格 | ~1.5-2亿欧元 | ~3.5亿欧元 |
| 每小时晶圆数 | ~200+ wph | ~185 wph（初期） |
| 首次商用 | 2019年 | 2025-2026年 |

### 高NA EUV的关键影响

1. **减少多重曝光**：标准EUV在2nm及以下需要SADP/SAQP（自对准双/四重图案化），高NA EUV可以单次曝光完成
2. **降低工艺步骤**：减少多重曝光意味着更少的工艺步骤，可能改善良率
3. **视场减半的代价**：高NA EUV的视场减半，意味着大芯片需要拼接（stitching），增加工艺复杂度
4. **成本上升**：单台价格翻倍，但减少了多重曝光的工艺成本

### ASML的垄断地位

ASML是全球唯一能制造EUV光刻机的公司。这一垄断地位在可预见的未来（至少到2030年）不会改变：
- EUV光刻机涉及10万+零部件，供应链横跨全球
- 30+年的技术积累和know-how
- 高NA EUV的研发投入超过200亿欧元
- 中国的国产EUV替代至少落后10-15年

## 四、晶圆经济学分析

### N5 → N3 → N2 价格递增但每晶体管成本改善

| 制程节点 | 晶圆价格（估计） | 晶体管密度（MTr/mm²） | 每晶体管成本 | 相比上代 |
|----------|----------------|----------------------|-------------|---------|
| N5 (5nm) | ~$16,000 | ~170 | 基准 | — |
| N3 (3nm) | ~$20,000 | ~290 | 约-15% | 密度↑70% > 价格↑25% |
| N2 (2nm) | ~$30,000 | ~450+ | 约-5% | 密度↑55% > 价格↑50% |

**关键洞察**：
- 晶圆价格从N5到N2几乎翻倍（$16K → $30K）
- 但每晶体管成本仍然在下降，因为密度提升更快
- N2的每晶体管成本改善幅度（~5%）明显小于N3（~15%），表明摩尔定律的经济回报在递减
- 只有最高价值的芯片（Apple A系列、NVIDIA GPU、高端服务器CPU）才能承受N2的价格

### 为什么设备商是确定性最高的受益者

不管TSMC、Intel还是Samsung赢得代工竞争，他们都需要：
- ASML的EUV光刻机（垄断）
- Applied Materials的沉积设备
- Lam Research的刻蚀设备
- KLA的检测量测设备
- TEL（东京电子）的全流程设备

**这是"淘金热卖铲子"的教科书案例。**

## 五、三大代工厂对比

| 代工厂 | GAA节点 | 背面供电 | 高NA采用 | 良率 | 首批客户 |
|--------|---------|---------|---------|------|---------|
| **TSMC** | N2 (2025Q4) | A16 (2026-2027) | 2027-2028 | ~70% | Apple, NVIDIA, Qualcomm |
| **Intel** | 18A (2026) | 18A (同时) | 2027（首个） | 较低（估计<60%） | 内部芯片 + Microsoft, Qualcomm |
| **Samsung** | SF2 (2025) | SF1.4 | 2028 | 50-70% | Qualcomm, Google Tensor |

### 竞争格局判断

1. **TSMC领先地位稳固**：N2良率70%+，客户锁定Apple和NVIDIA，A16背面供电跟进。最可能保持份额领先。
2. **Intel代工复兴的关键窗口**：18A如果良率能追到60%+，配合PowerVia的差异化，可能吸引部分客户。但连续延迟已经损害了客户信心。
3. **Samsung的GAA先发优势尚未转化为商业优势**：虽然最早引入GAA（SF3），但良率和客户关系落后于TSMC。
