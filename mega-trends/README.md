## 目录结构

```

├── 00_横向风险因子矩阵.md              ← 跨议题风险因子跟踪

│   └── issue_template.md
└── 01-21/                            ← 21个议题，连续编号
    ├── README.md                      概述、核心判断、关键指标

```

## 新增议题流程

1. 复制 `_template/issue_template.md` 创建新文件夹（编号从22递增）
2. 填写 README.md（概述、核心判断、关键指标）
3. 建立 timeline.md（已发生的关键事件）
4. 按需创建 technical_analysis.md 和 economic_impact.md
5. 在本文件索引中添加条目
6. 定期在 `updates/` 下追加最新进展
