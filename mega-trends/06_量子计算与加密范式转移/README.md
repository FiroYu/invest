## 一句话判断
> 量子计算正从实验室走向商业化（2025年收入$10-40亿），但实用化仍需5-15年。投资逻辑分三层：**后量子密码学(PQC)迁移**是确定性最高的主题（NIST标准已发布，全球金融系统必须更换加密基础设施）；量子模拟/药物发现是中期机会；通用量子计算和加密破解是远期投机。与AI是互补而非替代关系。

## 为什么重要
- **人类影响**：量子计算一旦破解RSA/ECC，全球金融系统、互联网安全、军事通信将面临系统性重构
- **经济影响**：McKinsey预测2035年量子经济影响$7200亿-$1万亿，但当前实际收入仅$10-40亿
- **时间敏感度**：2025年被McKinsey定义为"量子元年"；NIST已发布PQC标准，迁移窗口已打开

## 核心争议点

### 正方（量子时代即将到来）
- Google Willow实现100个逻辑量子比特，纠错突破里程碑
- IonQ收入增长202%，客户从实验室扩展到企业
- NIST发布PQC标准，全球迁移不可逆
- "先存储后解密"攻击已在进行，时间紧迫

### 反方（过度炒作）
- 量子纠错仍未解决从100→数百万的工程鸿沟
- 至今无令人信服的商业量子优势案例
- 经典计算（GPU/AI加速器）进步更快
- 纯量子股估值远超收入支撑（IonQ市值$245亿 vs 收入$1.3亿）
- 本源量子路线图承诺1024比特实际仅72比特

### 当前共识
- PQC迁移是确定性事件，与量子计算何时实用化无关
- 量子计算与AI是互补关系（量子加速特定计算，AI优化量子纠错）
- 纯量子概念股处于高度投机阶段
