# CIO裁决：议题#19 「AI对SaaS商业模式的颠覆性冲击」
## 裁决日期：2026年6月5日

---

## 一、核心裁决

### 议题评级：**「通过」→ 从「有条件通过」升级**

**理由**：

上一轮（2026.05.25）维持「有条件通过」，条件是「分化加剧——有护城河的SaaS可能幸存，纯水平SaaS面临生存级威胁」。

本轮确认以下关键催化剂：
1. **Salesforce Q1 FY27财报（5/27）** 提供了迄今为止最清晰的AI转型数据——Agentforce $1.2B ARR (+205% YoY)，但全年营收指引仅$45.9B-$46.2B（+10-11%），说明AI增量收入被传统业务减速部分抵消
2. **ServiceNow Q1 2026（4/22）** 业绩全面beat但利润率指引下调导致股价跌17%，证明市场对SaaS利润率极度敏感
3. **Palantir Q1 2026（5/4）** $1.63B营收+85% YoY，AI原生vs传统SaaS的分化从未如此鲜明
4. **水平SaaS 70-80%跌幅（HubSpot/Atlassian/Figma）** 确认了我们的分层判断
5. **Klarna企业级AI替代SaaS案例** 将威胁从「理论」升级为「正在发生的企业决策」

**「条件」仍然存在，但方向更明确**：不是所有SaaS同比例受创。我们不升级为「强力通过」的原因：
- Salesforce证明了龙头转型的可能性
- 部分垂直SaaS（安全、EDA、可观测性）反而受益于AI
- AI推理成本下降速度可能超预期

## 二、头寸决策

### 2.1 当前推荐配置

| 方向 | 标的 | 仓位 | 逻辑 |
|------|------|------|------|
| **做空** | HUBS (HubSpot) | 策略仓15% | 中端CRM+营销，AI替代最直接靶子，-70-80%已跌但仍有下行空间 |
| **做空** | TEAM (Atlassian) | 策略仓10% | 协作工具壁垒被AI侵蚀，-70-80%已跌但基本面恶化持续 |
| **做空** | CRM (Salesforce) | 观察仓5% | AI转型有望但估值仍偏高，EV/Rev 6.5x vs 目标4-5x |
| **做多** | PLTR (Palantir) | 核心仓8% | AI原生平台最纯标的，+85% YoY增速，但估值极高需设止损 |
| **做多** | DDOG (Datadog) | 策略仓8% | 可观测性因AI应用增加而受益，Usage-based不受定价转型冲击 |
| **做多** | SNPS (Synopsys) | 策略仓8% | EDA工具，AI芯片设计需求增长确定，35-40x PE合理 |
| **做多** | CRWD (CrowdStrike) | 策略仓8% | AI安全需求增长，安全SaaS有合规壁垒 |

**总仓位：62%，其中做空30% + 做多32%**

### 2.2 不做行动列表

| 标的 | 原因 |
|------|------|
| NOW (ServiceNow) | 护城河太深（ITSM工作流+77.5%毛利率），做空风险高 |
| WDAY (Workday) | -26-38% YTD已跌，但HCM替换是慢变量，做空时间成本高 |
| ADBE (Adobe) | 创意工具护城河复杂，AI竞争者（Canva/Midjourney）冲击速度不确定 |
| SNOW (Snowflake) | 已是Usage-based不受定价转型冲击，但AI产品毛利率<75%是隐忧 |
| MNDY/ASAN | 已跌足够多(70-80%)且市值小，空头拥挤，squeeze风险 |
