# 议题29：AI对软件SaaS商业模式的颠覆性冲击 — 时间线

## 2010-2019：SaaS黄金时代

| 时间 | 事件 |
|------|------|
| 2010-2015 | SaaS模式全面取代传统软件许可，Salesforce/ServiceNow/Workday高速增长 |
| 2014 | Salesforce营收突破$50亿，SaaS商业模式被全面验证 |
| 2015 | SaaS公司平均EV/Revenue达10-15x，市场给予极高估值 |
| 2016 | Slack/Twilio等新一代SaaS公司IPO，拓展SaaS边界 |
| 2017 | AWS/Azure/GCP推动云原生SaaS爆发 |
| 2018 | Adobe成功转型SaaS（从Creative Suite到Creative Cloud），传统软件转SaaS范例 |
| 2019 | HubSpot营收突破$10亿，中端SaaS市场崛起 |
| 2019 | SaaS行业平均毛利率达80%+，被称为"史上最好的商业模式" |

## 2020-2021：疫情泡沫期

| 时间 | 事件 |
|------|------|
| 2020.03 | COVID-19推动远程办公，SaaS需求爆发 |
| 2020 | Zoom营收增长326%，SaaS成为"疫情赢家" |
| 2020 | SaaS公司平均EV/Revenue飙升至20-30x |
| 2020.12 | Salesforce以$277亿收购Slack——SaaS行业整合信号 |
| 2021 | Snowflake IPO首日市值$800亿+，SaaS估值泡沫顶峰 |
| 2021 | 全球SaaS行业总市值突破$8万亿 |
| 2021 | "软件吞噬世界"成为共识，SaaS被视为不可战胜的商业模式 |

## 2022-2023：估值修正期

| 时间 | 事件 |
|------|------|
| 2022.01 | 美联储加息，高估值SaaS率先暴跌，BVP Cloud指数跌60%+ |
| 2022 | SaaS公司平均EV/Revenue从25x降至5-8x |
| 2022.11 | ChatGPT发布——AI革命开始，但市场尚未意识到对SaaS的冲击 |
| 2023.03 | GPT-4发布，AI编程能力飞跃 |
| 2023.03 | Salesforce宣布"Einstein GPT"——首个大型SaaS公司拥抱AI |
| 2023.05 | Microsoft Copilot发布——AI功能开始嵌入Office 365 |
| 2023 | SaaS公司开始大规模裁员（Salesforce裁员10%/Meta裁员/Google裁员） |
| 2023.Q3 | "AI溢价"叙事形成——市场认为AI是SaaS的增量收入来源 |
| 2023 | HubSpot股价在"AI溢价"叙事下上涨50%+ |

## 2024-2025：AI冲击显现期（当前阶段）

| 时间 | 事件 |
|------|------|
| 2024.Q1 | Salesforce推出Einstein Copilot，定价$500/用户/月——市场质疑过高 |
| 2024.02 | 德国SAP宣布全面AI化，但AI推理成本开始侵蚀毛利率 |
| 2024.Q1 | SaaS公司AI功能COGS占比上升，多家公司毛利率下降1-3个百分点 |
| 2024.06 | ServiceNow推出Now Assist AI功能，定价premium 30%——客户接受度一般 |
| 2024.H1 | AI编程工具（Cursor/GitHub Copilot）开发者采用率突破30% |
| 2024.08 | Gartner报告：15%的中端企业正在评估AI自建替代SaaS订阅 |
| 2024.Q3 | HubSpot流失率上升至8%+（历史水平5-6%），AI自建替代开始显现 |
| 2024.11 | 多家SaaS公司Q3财报显示：AI功能渗透率高但收入增量低于预期 |
| 2025.Q1 | "AI溢价"叙事开始破裂——SaaS公司AI功能定价承压 |
| 2025.03 | Salesforce调整AI定价策略，从固定收费转向usage-based |
| 2025.Q1 | 企业IT预算调查：SaaS支出增速从12%降至6%，AI基础设施支出增速30%+ |
| 2025.05 | SaaS行业平均毛利率降至75%（从2022年的82%） |
| 2025.H1 | AI编程工具采用率达40%+，中端企业自建案例增加 |

## 2026-2028：结构性重估期（预期）

| 时间 | 预期事件 |
|------|----------|
| 2026 | SaaS行业平均毛利率降至70%以下（AI推理成本持续计入COGS） |
| 2026 | Seat-based收入占比降至60%以下（usage-based占比上升） |
| 2026 | 3-5家中端SaaS公司因AI自建替代而大幅下修业绩指引 |
| 2027 | AI编程使企业自建软件成本降至SaaS订阅的50-70%（中端市场） |
| 2027 | Salesforce/ServiceNow等龙头增长放缓至10%以下 |
| 2027 | SaaS行业EV/Revenue中枢降至3-5x（从2021年高峰25x） |
| 2028 | AI原生软件公司（如Cursor/Midjourney等）颠覆部分SaaS品类 |
| 2028 | "软件订阅"不再是默认商业模式，usage-based/outcome-based成为主流 |
| 2028 | 至少1家大型SaaS公司因AI冲击被收购或大幅裁员 |

## 关键观察节点

- **2025-2026**：SaaS公司毛利率是否持续下降——AI成本侵蚀的量化证据
- **2025-2026**：Seat-based收入占比变化——定价模式转型是否加速
- **2026**：中端企业自建替代SaaS的实际案例——威胁是否变为现实
- **2026-2027**：Salesforce/ServiceNow增速是否降至10%以下——龙头增长见顶信号
- **持续关注**：AI编程工具能力提升速度——自建可行性提升的关键指标
