# 稳定币 / 数字货币基础设施 — 技术分析

## 1. 稳定币分类

### 1.1 法币抵押稳定币（当前主流）

| 稳定币 | 发行方 | 市值（2025） | 储备资产 | 结算网络 | 合规状态 |
|--------|--------|-------------|---------|---------|---------|
| USDT | Tether | ~$140B+ | 国库券、商业票据、公司债、BTC | 以太坊、Tron、Solana等 | 灰色地带，GENIUS后需调整 |
| USDC | Circle (CRCL) | ~$60B+ | 国库券、现金存款 | 以太坊、Solana、Base等 | 高度合规，MiCA合规 |
| PYUSD | PayPal (PYPL) | ~$1B+ | 国库券、现金存款 | 以太坊、Solana | 高度合规 |
| FDUSD | First Digital | ~$3B+ | 国库券、现金存款 | 以太坊、Solana | 香港监管 |
| USDS (原DAI) | Sky (原MakerDAO) | ~$5B+ | 国库券、USDC、ETH | 以太坊 | 去中心化治理，逐步合规化 |

### 1.2 加密抵押稳定币

| 稳定币 | 机制 | 风险特征 |
|--------|------|---------|
| DAI（原版） | ETH/WBTC超额抵押+智能合约清算 | 去中心化，但依赖加密资产价格稳定性 |
| LUSD | ETH抵押（无治理） | 最小化治理风险，但规模有限 |

### 1.3 算法稳定币（高风险）

| 稳定币 | 状态 | 教训 |
|--------|------|------|
| UST（Terra/Luna） | 2022年5月崩盘，$40B归零 | 算法稳定币的"死亡螺旋"风险——一旦信心动摇，无法阻止挤兑 |
| FRAX | 部分算法→转向完全抵押 | 行业从UST崩盘中吸取教训，纯算法模式已基本被放弃 |

**关键洞察**：UST崩盘后，市场共识明确转向"完全抵押"模式。GENIUS法案和MiCA法规均要求1:1法币储备，实际上将算法稳定币排除在主流之外。

## 2. GENIUS法案核心条款

### 2.1 法案概览
- **全称**：Guiding and Establishing National Innovation for U.S. Stablecoins Act
- **签署时间**：2025年7月
- **最终规则生效**：预计2026年中

### 2.2 核心要求

| 条款 | 要求 | 影响 |
|------|------|------|
| 储备要求 | 1:1美元资产支持（国库券、现金存款等） | 确保赎回能力，消除运行风险 |
| 发行人资质 | 必须是受监管的金融机构或其子公司 | 提高准入门槛，淘汰不合格发行人 |
| 监管框架 | 联邦（OCC注册）或州（经批准的州监管）双轨制 | 平衡创新与监管 |
| AML/CFT | 完整的反洗钱和反恐融资合规 | 将稳定币纳入传统金融合规体系 |
| 审计与披露 | 定期审计储备，公开披露储备构成 | 解决Tether长期以来的透明度问题 |
| 消费者保护 | 发行人在破产时必须优先保障稳定币持有人的赎回权 | 消除"稳定币发行人挪用储备"的风险 |
| 禁止算法稳定币 | 不允许发行不依赖法币储备的算法稳定币 | 直接回应UST崩盘 |

### 2.3 战略意义

**对美元霸权**：
- GENIUS法案将美元稳定币纳入监管框架，实际上是在数字世界中扩展美元的影响力
- 全球用户通过持有美元稳定币，间接使用美元，强化美元的全球储备货币地位
- 这是对中国数字人民币（e-CNY）和欧盟数字欧元的战略回应

**对国库券市场**：
- 1:1美元储备要求意味着稳定币发行人需要购买大量国库券
- 当前$290B市值 → $290B国库券需求
- 随着稳定币市值增长至$1T+，国库券需求可达$900B-$1T
- 这是结构性需求，不会因利率变化而减少

## 3. MiCA vs GENIUS对比

| 维度 | 欧盟MiCA | 美国GENIUS |
|------|---------|-----------|
| 生效时间 | 2024年6月（稳定币）/ 2024年12月（全面） | 2025年7月签署，2026年中最终规则 |
| 储备要求 | 1:1法币/高质量流动资产 | 1:1美元资产 |
| 发行主体 | 信贷机构或电子货币机构 | 受监管金融机构或其子公司 |
| 监管机构 | EBA（欧盟银行管理局）+各国监管 | OCC（联邦）或州监管（双轨） |
| 算法稳定币 | 严格限制 | 完全禁止 |
| 日赎回限额 | 当规模过大时可施加日赎回限额 | 无明确限制 |
| 稳定币类型分类 | EMT（电子货币代币）+ ART（资产参考代币） | 统一为"支付稳定币" |
| 市场影响 | 欧元稳定币发展较慢，USDT/USDC主导 | 银行入场预期强，美元稳定币优势扩大 |

**关键差异**：GENIUS更加灵活和市场化，MiCA更加审慎和限制性。GENIUS的双轨监管设计（联邦+州）借鉴了美国银行监管的成功经验。

## 4. 技术栈：从发行到支付

### 4.1 完整技术架构

```
┌─────────────────────────────────────────────┐
│              应用层                           │
│   钱包App、支付网关、商户API、DeFi协议         │
├─────────────────────────────────────────────┤
│              合规层                           │
│   KYC/AML、链上监控、旅行规则合规、制裁筛查      │
├─────────────────────────────────────────────┤
│              结算层                           │
│   以太坊(L1/L2)、Solana、Base、Arbitrum等     │
├─────────────────────────────────────────────┤
│              发行与托管层                      │
│   稳定币智能合约、储备管理、铸币/销毁机制        │
├─────────────────────────────────────────────┤
│              储备资产层                        │
│   国库券、银行存款、货币市场基金                 │
└─────────────────────────────────────────────┘
```

### 4.2 各层关键参与者

| 层次 | 关键参与者 | 技术/产品 |
|------|-----------|---------|
| 储备资产层 | BlackRock、Fidelity、State Street | 国库券基金、货币市场基金 |
| 发行与托管层 | Circle、Tether、Fireblocks、BitGo | 智能合约、机构托管 |
| 结算层 | Ethereum、Solana、Base、Arbitrum | 公链/L2网络 |
| 合规层 | Chainalysis、Elliptic、TRM Labs | 链上AML/KYC工具 |
| 应用层 | MetaMask、Phantom、Stripe、Shopify | 钱包、支付网关 |

### 4.3 结算网络对比

| 网络 | TPS（理论） | 最终确认时间 | 平均交易费 | 适用场景 |
|------|-----------|------------|----------|---------|
| 以太坊L1 | ~15 | ~12秒 | $1-5 | 高价值结算 |
| Solana | ~65,000 | ~0.4秒 | <$0.01 | 支付/高频交易 |
| Base (L2) | ~2,000+ | ~2秒 | <$0.01 | 通用支付 |
| Arbitrum (L2) | ~2,000+ | ~2秒 | <$0.01 | DeFi/通用 |
| Ethereum (Danksharding后) | ~100,000+ | — | — | 未来 |

**关键洞察**：Solana在支付场景的TPS和成本已接近传统支付网络（Visa ~65,000 TPS），但去中心化程度和稳定性仍有差距。Layer 2方案（Base、Arbitrum）在成本和安全性之间提供了更好的平衡。

## 5. 对国库券市场的影响机制

### 5.1 传导路径

```
稳定币需求增长 → 发行人增发稳定币 → 需要购买更多美元储备资产
→ 购买国库券（最主要储备资产）→ 国库券需求增加
→ 国库券收益率可能下行（价格上行）→ 影响全球无风险利率锚
```

### 5.2 需求冲击量化

| 稳定币市值 | 国库券需求（假设80%配置） | 占国库券市场比例 |
|-----------|----------------------|----------------|
| $290B（当前） | ~$232B | ~1-2% |
| $500B（2027年预期） | ~$400B | ~3% |
| $1T（2029年预期） | ~$800B | ~5% |
| $2T（2030s可能） | ~$1.6T | ~8-10% |

**关键影响**：
- 当前阶段（$290B）：影响有限，可忽略
- 中期（$500B-$1T）：开始对短端国库券收益率产生可观测的下行压力
- 长期（$1T+）：可能成为国库券市场的重要结构性需求来源，影响收益率曲线形态

### 5.3 与美联储货币政策的交互
- 稳定币增加对国库券的需求，可能在加息周期中部分抵消收益率上行压力
- 稳定币吸走银行存款 → 银行放贷能力下降 → 可能影响货币乘数
- 美联储可能需要在货币政策框架中考虑稳定币对货币供应的影响
