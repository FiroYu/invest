# 议题09：稳定币与数字货币基础设施革命

## 核心判断

GENIUS法案于2025年7月签署成为法律，标志着合规稳定币从灰色地带正式进入主流金融体系。稳定币总市值从5年前的~$5B飙升至~$290B（增长58倍），USDC季度链上交易量达$11.9T（同比+247%）。稳定币可能对国库券产生$900B级别的需求冲击，并从根本上重构全球支付基础设施。

**范式转移的核心逻辑：**
- **监管合法化**：GENIUS法案将稳定币发行人纳入联邦金融监管框架，消除合规不确定性
- **使用场景爆发**：从加密交易工具扩展到跨境支付、B2B结算、消费支付
- **国库券需求冲击**：稳定币发行人必须持有1:1美元储备（主要是国库券），可能创造$900B级别的新结构性需求
- **银行入场**：GENIUS实施后，大型银行（JPMorgan、Bank of America等）可能发行自有稳定币

## 争议与不确定性

### 银行体系存款外流风险
- 稳定币可能吸走商业银行存款，影响银行放贷能力
- 美联储和FDIC对银行参与稳定币的态度仍在演变
- GENIUS法案要求1:1美元支持，但允许的储备资产范围仍在讨论中

### 技术基础设施成熟度
- 区块链结算速度和成本仍不及传统支付网络（Visa每秒6.5万笔）
- Layer 2解决方案（Solana、Base、Arbitrum）正在缩小差距
- 合规工具（链上AML/KYC）仍在早期阶段

### 地缘政治维度
- 美元稳定币的全球扩张强化美元霸权，部分国家可能抵制
- 中国数字人民币（e-CNY）作为替代方案的进展
- 欧盟MiCA框架下的欧元稳定币发展

### Tether的合规风险
- USDT（Tether）占稳定币市场~60%份额，但合规透明度长期受质疑
- GENIUS法案可能迫使Tether大幅调整运营模式
- 若Tether出现问题，可能引发短期市场动荡
